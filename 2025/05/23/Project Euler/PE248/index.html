PE248

Posted on 2025-05-23 In Project Euler

题目

The first number n for which ϕ(n) = 13! is 6227180929. Find the 150, 000th such number.

找到欧拉函数值是13!的第150000个数

题解

ϕ(n) = 13! = 6227020800

的第 150 000 个最小的正整数 n。 由于欧拉函数不是单调的，不能直接逆推，因此必须通过构造所有满足 ϕ(n) = 6227020800 的数，然后排序取第 150000 个。

核心思路

1. 分析：ϕ(n) = ϕ(p1e1) × ϕ(p2e2) × …

欧拉函数是 积性函数，对所有互素的因子满足：

ϕ(pe) = pe − pe − 1 = pe − 1 ⋅ (p − 1)

所以:

ϕ(n) = ∏ipiei − 1 ⋅ (p − 1)

逐步解释

第 0 步：预设和基础定义

目标是 ϕ(n) = M = 13!，我们要找第 K 个这样的 n。

第 1 步：构造所有可能的素因子（包括大素数）

这些是 13! 的素因子。

然后我们生成所有正整数 d 使得 d ∣ M，并尝试 p = d + 1，因为：

若 p 是素数，且 ϕ(p) = p − 1 ∣ M，那么 p 是一个可能的因子。

这些大素数是将来构造候选 n 的关键。

第 2 步：为每个素数构造「块」(block)

举例来说，对于 p = 3，你会得到：

  • (ϕ(31), 31) = (2, 3)
  • (ϕ(32), 32) = (6, 9)
  • (ϕ(33), 33) = (18, 27)
  • 等等，只要 ϕ(pk) ∣ M

第 3 步：合并所有 blocks，构造所有 (ϕ(n), n) 对

这一步是算法的关键。

我们一开始只有 (1, 1)，然后不断将它与每个素数的所有幂次组合合并。并且：

✅ 只保留能整除 M 的 ϕ，否则不可能组合出 ϕ(n) = M。

这样最终 current 中包含了所有可能组合出来的合法 (ϕ(n), n)。

第 4 步：筛选出 ϕ(n) = M，排序取第 K 个

最后我们从中筛选出所有 ϕ(n) = 13! 的 n，并排序，取出第 K 个：

总结关键点

步骤 目的
构造约数 d 得到可能的大素数 p = d + 1 确保不会漏掉构造 ϕ(n) 的质因子
为每个素数构造块 (ϕ(pk), pk) 利用欧拉函数积性结构
合并块，仅保留 ϕ ∣ M 避免爆炸增长并保证合法性
最终筛选 ϕ(n) = M，排序取第 K 个 n 得到目标解
