题意

题目链接：3895. Count Digit Appearances

解题思路

对每个数 x 做十进制拆位：

  1. 取当前最低位 x % 10，若等于 digit 则计数加一。
  2. 令 x /= 10，继续处理下一位。
  3. 直到 x == 0。

题意

题目链接：3896. Minimum Operations to Transform Array into Alternating Prime

  1. 偶数下标元素为质数。
  2. 奇数下标元素为非质数。

求最小操作次数。

解题思路

每个位置的修改互不影响，可以逐位独立计算最小代价后求和。
