3936 - Minimum Swaps to Move Zeros to End

题意

题目链接：Minimum Swaps to Move Zeros to End

给定一个整数数组 nums。每次操作可以选择两个不同位置 i 和 j，交换 nums[i] 与 nums[j]。要求用最少操作次数，使得数组中的所有 0 都移动到末尾。

解题思路

设数组中 0 的数量为 z。最终数组的最后 z 个位置必须都是 0，因此只需要观察前 n - z 个位置。

这 n - z 个位置中每出现一个 0，就意味着它必须与后缀中的一个非零数交换一次。一次交换可以同时修正一个前缀 0 和一个后缀非零数，所以最少交换次数就是前 n - z 个位置中 0 的数量。

时间复杂度：，需要遍历数组统计零的数量，再扫描一次前缀。

空间复杂度：，只使用了常数个额外变量。

3937 - Minimum Operations to Make Array Modulo Alternating I

题意

题目链接：Minimum Operations to Make Array Modulo Alternating I

给定数组 nums 和整数 k。每次操作可以让任意一个元素加一或减一。要求存在两个不同的整数 x 和 y，满足：

  • 所有偶数下标 i：nums[i] % k == x
  • 所有奇数下标 i：nums[i] % k == y

返回最少操作次数。

解题思路

先看单个元素。若当前余数为 r，目标余数为 t，那么最少操作次数就是模 k 环上的最短距离：

由于 k <= 100，可以直接枚举偶数下标统一变成的余数 x，以及奇数下标统一变成的余数 y，并要求 x != y。

3938 - Maximum Path Intersection Sum in a Grid

题意

题目链接：Maximum Path Intersection Sum in a Grid

给定一个 m x n 的整数矩阵 grid。

玩家 1 从左上角 (0, 0) 出发，只能向右或向下走，终点是右下角 (m - 1, n - 1)。 玩家 2 从左下角 (m - 1, 0) 出发，只能向右或向上走，终点是右上角 (0, n - 1)。

如果某个格子同时属于两条路径，那么它是共享格子。要求返回所有共享格子权值和的最大值。

解题思路

先刻画交集形状。设两个共享格子分别为 (a, b) 和 (c, d)，且 b < d。由于玩家 1 的行号只会不减，玩家 2 的行号只会不增，因此必须同时有 a <= c 和 a >= c，从而 a == c。这说明：不同列的共享格子只能位于同一行。

同理，不同行的共享格子只能位于同一列。

因此，两条路径的交集只能是：

  • 某一行上的连续段
  • 某一列上的连续段
  • 一个内部单点

其中长度为 1 的交集只能发生在内部格子；边界格子无法单独作为交集，因为两条路径若在边界只共享一个格子，无法完成一次真正的横纵交叉。

于是问题就变成了三类候选的最大值：

  1. 每一行中，长度至少为 2 的最大连续子段和；
  2. 每一列中，长度至少为 2 的最大连续子段和；
  3. 每个内部格子单独作为交集的值。

前两类可以用一个变形的 Kadane 算法处理：current 表示必须以当前位置结尾、且长度至少为 2 的最大连续子段和，转移时只需在“接在前面继续”和“从当前两个位置重新开始”之间取较大值。

3939 - Count Non Adjacent Subsets in a Rooted Tree

题意

题目链接：Count Non Adjacent Subsets in a Rooted Tree

给定一棵以 0 为根的有根树，parent[i] 表示节点 i 的父节点。每个节点 i 有权值 nums[i]，再给定整数 k。统计非空节点子集的数量，满足：

  • 选中节点的权值和能被 k 整除；
  • 不能同时选择一条边的两个端点，也就是不能同时选择父子节点。

返回方案数对 10^9 + 7 取模后的结果。

解题思路

这是树形 DP。对每个节点 u，维护两类状态：

  • f_0[u][r]：在 u 的子树中选择一些节点，且不选 u，总和模 k 为 r 的方案数；
  • f_1[u][r]：在 u 的子树中选择一些节点，且选 u，总和模 k 为 r 的方案数。

初始化时：

  • 不选 u，空集合法，所以 f_0[u][0] = 1
  • 只选 u，所以 f_1[u][nums[u] % k] = 1

合并儿子 v 时分两种情况：

  1. 如果当前不选 u，那么 v 可以选，也可以不选，因此儿子子树的可用方案数是 f_0[v] + f_1[v]。
  2. 如果当前选了 u，那么 v 不能选，只能使用 f_0[v]。

把两个余数数组合并时，方案数相乘，余数相加取模，即新的余数是 (i + j) % k。

所有儿子合并完后，根节点的答案是 f_0[0][0] + f_1[0][0]，但其中 f_0[0][0] 包含空集，所以最后需要减去 1。
