3955 - Valid Binary Strings With Cost Limit

题意

题目链接：Valid Binary Strings With Cost Limit

长度为 n 的二进制字符串 s 的代价定义为所有满足 s[i] == '1' 的下标 i 之和，下标从 0 开始。

若一个二进制字符串同时满足以下条件，则称它合法：

  • 不包含连续的两个 '1'；
  • 代价不超过 k。

返回所有长度为 n 的合法二进制字符串，顺序任意。

  • 1 <= n <= 12
  • 0 <= k <= n * (n - 1) / 2

解题思路

从左到右构造字符串。DFS 状态记录：

  • i：当前准备填写的位置；
  • c：已经放置的所有 '1' 的下标之和；
  • pre：前一个字符是否为 '1'。

在位置 i：

  1. 始终可以填写 '0'，代价和前一位状态分别变为 c 和 0；
  2. 只有当 pre == 0 且 c + i <= k 时才能填写 '1'。

第二个分支同时完成了两种剪枝：!pre 保证没有连续的 '1'，c + i <= k 保证代价不会超出限制。因为后续新增代价均为非负数，一旦当前代价超过 k，该分支不可能重新变得合法。

当 i == n 时，当前字符串已经满足所有约束，加入答案。

解题思路

1. WQS 二分的基本思想

WQS 二分也称 Alien Trick、带权二分或拉格朗日松弛。它适用于一类带有“恰好或至多选择 m 个对象”限制的最优化问题。

其直观含义是：暂时取消数量限制，但每选择一个对象都收取罚金 x。

  • 罚金较小时，多选一个子数组的代价较低，最优方案倾向于选择更多子数组；
  • 罚金较大时，多选一个子数组的代价较高，最优方案倾向于选择更少子数组；
  • 二分罚金，就可以找到选择数量跨过 m 的临界位置。
