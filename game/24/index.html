24 Game

Use 4 numbers and + - * / ( ) to make 24. Each number must be used exactly once.

Solver algorithm: DFS over pair selections and 6 operations. Recurse until one value remains and test whether it equals 24.
Allowed characters: digits, spaces, parentheses, and + - * /. Numbers 10-13 are treated as two-digit values.
