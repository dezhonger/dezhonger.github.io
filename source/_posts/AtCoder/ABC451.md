## A - illegal

**题意**

题目链接：[abc451_a](https://atcoder.jp/contests/abc451/tasks/abc451_a)

给一个只含小写字母的字符串 $S$，如果 $|S|$ 是 5 的倍数输出 `Yes`，否则输出 `No`。

**解题思路**

直接判断 `S.size() % 5 == 0`。

**代码**
```cpp
void solve() {
    string s;
    cin >> s;
    cout << ((int)s.size() % 5 == 0 ? "Yes\n" : "No\n");
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    while (T--) {
        solve();
    }
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(|S|)$。  
空间复杂度：$O(|S|)$。

**变形**
1. 把 5 改成任意 $k$，判断是否为 $k$ 的倍数。
2. 同时增加其他条件，例如首字符限制或长度区间限制。

## B - Personnel Change

**题意**

题目链接：[abc451_b](https://atcoder.jp/contests/abc451/tasks/abc451_b)

有 $N$ 名员工，本学期在部门 $A_i$，下学期在部门 $B_i$。对每个部门 $j$，输出：

$$
\text{next}(j) - \text{now}(j)
$$

**解题思路**

每个员工对两个部门贡献：

- 对 $A_i$ 贡献 $-1$
- 对 $B_i$ 贡献 $+1$

累加即可。

**代码**
```cpp
void solve() {
    int n, m;
    cin >> n >> m;
    vector<int> d(m + 1, 0);
    for (int i = 0; i < n; i++) {
        int a, b;
        cin >> a >> b;
        d[a]--;
        d[b]++;
    }
    for (int j = 1; j <= m; j++) {
        cout << d[j] << '\n';
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    while (T--) {
        solve();
    }
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(N + M)$。  
空间复杂度：$O(M)$。

**变形**
1. 统计每个部门本学期和下学期的绝对人数。
2. 改成在线多次人员调整，要求每次修改后立即输出变化量。

## C - Understory

**题意**

题目链接：[abc451_c](https://atcoder.jp/contests/abc451/tasks/abc451_c)

初始空集合，处理 $Q$ 个操作：

- `1 h`：插入一棵高度为 `h` 的树
- `2 h`：删除所有高度 $\le h$ 的树

每次操作后输出当前树的数量。

**解题思路**

用 `multiset<int>` 维护当前所有高度。

- 插入：`st.insert(h)`
- 删除：`it = st.upper_bound(h)`，执行 `st.erase(st.begin(), it)`

`upper_bound(h)` 前面的元素正好是所有 $\le h$ 的元素。

**代码**
```cpp
void solve() {
    int q;
    cin >> q;
    multiset<int> st;

    while (q--) {
        int op, h;
        cin >> op >> h;
        if (op == 1) {
            st.insert(h);
        } else {
            auto it = st.upper_bound(h);
            st.erase(st.begin(), it);
        }
        cout << st.size() << '\n';
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    while (T--) {
        solve();
    }
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(Q \log Q)$（删除区间按元素总删除次数均摊）。  
空间复杂度：$O(Q)$。

**变形**
1. 删除区间 $[l, r]$ 的元素。
2. 额外支持查询第 $k$ 小，通常需要树状数组/线段树/PBDS。

## D - Concat Power of 2

**题意**

题目链接：[abc451_d](https://atcoder.jp/contests/abc451/tasks/abc451_d)

“好整数”定义为：把一个或多个 $2$ 的幂（十进制字符串）拼接后得到的整数。允许重复、允许重排。给定 $N$，求第 $N$ 小好整数，保证答案 $\le 10^9$。

**解题思路**

先列出所有 $\le 10^9$ 的 $2$ 的幂字符串作为可选块。然后 DFS 枚举所有拼接结果，剪枝条件是拼接后数值不超过 $10^9$。最后把所有结果排序去重，取第 $N$ 小。

**代码**
```cpp
vector<string> blocks;
vector<int> all_values;
const string LIM = "1000000000";

bool leq_limit(const string &s) {
    if (s.size() < LIM.size()) return true;
    if (s.size() > LIM.size()) return false;
    return s <= LIM;
}

int to_int(const string &s) {
    int x = 0;
    for (char c : s) x = x * 10 + (c - '0');
    return x;
}

void dfs(string &cur) {
    if (!cur.empty() && leq_limit(cur)) {
        all_values.push_back(to_int(cur));
    }
    for (const string &b : blocks) {
        if ((int)cur.size() + (int)b.size() > 10) continue;
        int old = (int)cur.size();
        cur += b;
        if (leq_limit(cur)) dfs(cur);
        cur.resize(old);
    }
}

void solve() {
    int n;
    cin >> n;

    blocks.clear();
    all_values.clear();

    LL x = 1;
    while (x <= 1000000000LL) {
        blocks.push_back(to_string(x));
        x <<= 1;
    }

    string cur;
    dfs(cur);

    sort(all_values.begin(), all_values.end());
    all_values.erase(unique(all_values.begin(), all_values.end()), all_values.end());

    cout << all_values[n - 1] << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    while (T--) {
        solve();
    }
    return 0;
}
```

**复杂度分析**

设枚举得到的候选数量为 $M$。  
时间复杂度：$O(M \log M)$。  
空间复杂度：$O(M)$。

**变形**
1. 把底层集合从 $2$ 的幂换成 $3$ 的幂或斐波那契数。
2. 改为统计区间 $[L, R]$ 内好整数个数。

## E - Tree Distance

**题意**

题目链接：[abc451_e](https://atcoder.jp/contests/abc451/tasks/abc451_e)

给出 $N$ 个点两两距离矩阵 $A_{i,j}$，判断是否存在一棵带权树，使得树上任意两点距离恰好等于该矩阵。

**解题思路**

把 $A$ 看成完全图边权。若原树存在，则该树边在对应割上是唯一最小边，因此原树必须等于该完全图的一棵 MST。做法：

1. 用 Prim 在距离完全图上构造一棵候选 MST。
2. 在候选树上从每个点 BFS/DFS 求到所有点距离。
3. 若所有距离都与输入矩阵一致，输出 `Yes`，否则 `No`。

**代码**
```cpp
int n;
vector<vector<LL>> a;
vector<vector<pair<int, LL>>> g;

bool bfs_check(int s) {
    vector<LL> d(n + 1, -1);
    vector<int> fa(n + 1, -1), q;
    q.push_back(s);
    fa[s] = 0;
    d[s] = 0;

    for (int i = 0; i < (int)q.size(); i++) {
        int u = q[i];
        for (auto [v, w] : g[u]) {
            if (v == fa[u]) continue;
            fa[v] = u;
            d[v] = d[u] + w;
            q.push_back(v);
        }
    }

    for (int i = 1; i <= n; i++) {
        if (d[i] != a[s][i]) return false;
    }
    return true;
}

void solve() {
    cin >> n;
    a.assign(n + 1, vector<LL>(n + 1, 0));

    for (int i = 1; i <= n; i++) {
        for (int j = i + 1; j <= n; j++) {
            cin >> a[i][j];
            a[j][i] = a[i][j];
        }
    }

    g.assign(n + 1, {});

    const LL INF = (LL)4e18;
    vector<LL> mn(n + 1, INF);
    vector<int> pre(n + 1, -1), vis(n + 1, 0);

    mn[1] = 0;
    for (int it = 1; it <= n; it++) {
        int u = -1;
        for (int i = 1; i <= n; i++) {
            if (!vis[i] && (u == -1 || mn[i] < mn[u])) u = i;
        }

        vis[u] = 1;
        if (pre[u] != -1) {
            g[u].push_back({pre[u], mn[u]});
            g[pre[u]].push_back({u, mn[u]});
        }

        for (int v = 1; v <= n; v++) {
            if (!vis[v] && a[u][v] < mn[v]) {
                mn[v] = a[u][v];
                pre[v] = u;
            }
        }
    }

    for (int s = 1; s <= n; s++) {
        if (!bfs_check(s)) {
            cout << "No\n";
            return;
        }
    }
    cout << "Yes\n";
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    while (T--) {
        solve();
    }
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(N^2)$。  
空间复杂度：$O(N^2)$。

**变形**
1. 允许引入未标号中间点时，问题会转向一般树度量判定。
2. 若要求输出树边，当前方案在通过校验后可直接输出 MST 边集。

## F - Make Bipartite 3

**题意**

题目链接：[abc451_f](https://atcoder.jp/contests/abc451/tasks/abc451_f)

初始无边图，按顺序加入 $Q$ 条边。每次加入后：

1. 判断图是否仍可二染色。
2. 若可行，求黑点数最小值；否则输出 `-1`。

**解题思路**

使用带奇偶关系的并查集（parity DSU）：

- $d[x]$ 表示 $x$ 到所在根的颜色异或值（0 同色，1 异色）。
- 每个连通块维护 $c_0/c_1$：与根同色和异色的点数。
- 连通块贡献是 $\min(c_0, c_1)$，全局答案为所有连通块贡献和。
- 加边 $(u,v)$ 约束是 $\operatorname{color}(u) \oplus \operatorname{color}(v) = 1$。

若同根且约束冲突，图从此不再二分。

**代码**
```cpp
struct DSU {
    vector<int> fa, sz, d, c0, c1;

    DSU(int n) {
        fa.resize(n + 1);
        sz.assign(n + 1, 1);
        d.assign(n + 1, 0);
        c0.assign(n + 1, 1);
        c1.assign(n + 1, 0);
        for (int i = 1; i <= n; i++) fa[i] = i;
    }

    int find(int x) {
        if (fa[x] == x) return x;
        int p = fa[x];
        fa[x] = find(p);
        d[x] ^= d[p];
        return fa[x];
    }
};

void solve() {
    int n, q;
    cin >> n >> q;

    DSU uf(n);
    LL ans = 0;
    bool ok = true;

    while (q--) {
        int u, v;
        cin >> u >> v;

        if (!ok) {
            cout << -1 << '\n';
            continue;
        }

        int ru = uf.find(u), rv = uf.find(v);
        int du = uf.d[u], dv = uf.d[v];

        if (ru == rv) {
            if ((du ^ dv) != 1) ok = false;
            cout << (ok ? ans : -1) << '\n';
            continue;
        }

        ans -= min(uf.c0[ru], uf.c1[ru]);
        ans -= min(uf.c0[rv], uf.c1[rv]);

        if (uf.sz[ru] < uf.sz[rv]) {
            swap(ru, rv);
            swap(du, dv);
        }

        int t = du ^ dv ^ 1;
        uf.fa[rv] = ru;
        uf.d[rv] = t;
        uf.sz[ru] += uf.sz[rv];

        if (t == 0) {
            uf.c0[ru] += uf.c0[rv];
            uf.c1[ru] += uf.c1[rv];
        } else {
            uf.c0[ru] += uf.c1[rv];
            uf.c1[ru] += uf.c0[rv];
        }

        ans += min(uf.c0[ru], uf.c1[ru]);
        cout << ans << '\n';
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    while (T--) {
        solve();
    }
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(Q \alpha(N))$。  
空间复杂度：$O(N)$。

**变形**
1. 同时支持删边时，通常需要可撤销并查集 + 线段树分治。
2. 若目标改成“最大黑点数”，每个连通块贡献改为 $\max(c_0, c_1)$。

## G - Minimum XOR Walk

**题意**

题目链接：[abc451_g](https://atcoder.jp/contests/abc451/tasks/abc451_g)

给定连通无向带权图，walk 的权值定义为经过边权的异或和。定义 $f(x,y)$ 为 $x,y$ 之间所有 walk 权值的最小值。对每组数据，统计有多少对 $x < y$ 满足 $f(x,y) \le K$。

**解题思路**

固定根并求每个点的树上前缀异或 $d[v]$。对每条边 $(u,v,w)$，构造环值：

$$
c = d[u] \oplus d[v] \oplus w
$$

这些 $c$ 张成环空间线性基 $S$。可证明：

$$
f(x,y) = \min_{s \in S} (d[x] \oplus d[y] \oplus s)
$$

将线性基规整后定义 $red(z)$ 为陪集中最小值，则

$$
f(x,y) = red(d[x]) \oplus red(d[y])
$$

设 $a[i] = red(d[i])$，问题变成统计 $a[x] \oplus a[y] \le K$ 的点对数量，用 01-Trie 在线统计即可。

**代码**
```cpp
struct LB {
    int bs[30];

    LB() { memset(bs, 0, sizeof(bs)); }

    void ins(int x) {
        for (int i = 29; i >= 0; i--) {
            if (((x >> i) & 1) == 0) continue;
            if (!bs[i]) {
                bs[i] = x;
                return;
            }
            x ^= bs[i];
        }
    }

    void fix() {
        for (int i = 0; i < 30; i++) {
            if (!bs[i]) continue;
            for (int j = 0; j < i; j++) {
                if ((bs[i] >> j) & 1) bs[i] ^= bs[j];
            }
        }
    }

    int red(int x) {
        for (int i = 29; i >= 0; i--) {
            if (bs[i] && ((x >> i) & 1)) x ^= bs[i];
        }
        return x;
    }
};

struct Trie {
    struct Node {
        int ch[2];
        int cnt;
        Node() {
            ch[0] = ch[1] = -1;
            cnt = 0;
        }
    };

    vector<Node> tr;

    Trie() {
        tr.clear();
        tr.push_back(Node());
    }

    void ins(int x) {
        int p = 0;
        tr[p].cnt++;
        for (int i = 29; i >= 0; i--) {
            int b = (x >> i) & 1;
            if (tr[p].ch[b] == -1) {
                tr[p].ch[b] = (int)tr.size();
                tr.push_back(Node());
            }
            p = tr[p].ch[b];
            tr[p].cnt++;
        }
    }

    int ask(int x, int k) {
        int p = 0;
        int ans = 0;
        for (int i = 29; i >= 0; i--) {
            if (p == -1) break;
            int xb = (x >> i) & 1;
            int kb = (k >> i) & 1;
            if (kb == 1) {
                int q = tr[p].ch[xb];
                if (q != -1) ans += tr[q].cnt;
                p = tr[p].ch[xb ^ 1];
            } else {
                p = tr[p].ch[xb];
            }
        }
        if (p != -1) ans += tr[p].cnt;
        return ans;
    }
};

void solve() {
    int n, m, k;
    cin >> n >> m >> k;

    vector<int> u(m), v(m), w(m);
    vector<vector<pair<int, int>>> g(n + 1);

    for (int i = 0; i < m; i++) {
        cin >> u[i] >> v[i] >> w[i];
        g[u[i]].push_back({v[i], w[i]});
        g[v[i]].push_back({u[i], w[i]});
    }

    vector<int> d(n + 1, 0), vis(n + 1, 0);
    queue<int> q;
    q.push(1);
    vis[1] = 1;
    while (!q.empty()) {
        int x = q.front();
        q.pop();
        for (auto [y, z] : g[x]) {
            if (vis[y]) continue;
            vis[y] = 1;
            d[y] = d[x] ^ z;
            q.push(y);
        }
    }

    LB lb;
    for (int i = 0; i < m; i++) {
        int c = d[u[i]] ^ d[v[i]] ^ w[i];
        if (c) lb.ins(c);
    }
    lb.fix();

    vector<int> a(n + 1);
    for (int i = 1; i <= n; i++) a[i] = lb.red(d[i]);

    Trie tr;
    LL ans = 0;
    for (int i = 1; i <= n; i++) {
        ans += tr.ask(a[i], k);
        tr.ins(a[i]);
    }

    cout << ans << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    cin >> T;
    while (T--) {
        solve();
    }
    return 0;
}
```

**复杂度分析**

时间复杂度：$O((N + M) \cdot 30)$。  
空间复杂度：$O(N + M)$。

**变形**
1. 只问单对点最小 XOR walk 时，可省去 Trie，仅做一次线性基最小化。
2. 把比较条件改成 $= C$ 时，可转成哈希/Trie 统计异或等值对。
