## A - Gothec

**题意**

题目链接：[abc452_a](https://atcoder.jp/contests/abc452/tasks/abc452_a)

给定日期 $(M,D)$，若它是五节句之一 `(1,7),(3,3),(5,5),(7,7),(9,9)`，输出 `Yes`，否则输出 `No`。

**解题思路**

直接常数次判断这 5 个日期。

**代码**
```cpp
void solve() {
    int m, d;
    cin >> m >> d;
    bool ok = (m == 1 && d == 7) || (m == 3 && d == 3) || (m == 5 && d == 5)
           || (m == 7 && d == 7) || (m == 9 && d == 9);
    cout << (ok ? "Yes\n" : "No\n");
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(1)$。空间复杂度：$O(1)$。

**变形**

1. 节日列表改为任意给定集合，可用 `set<pair<int,int>>`。
2. 多次查询时直接哈希判定。

## B - Draw Frame

**题意**

题目链接：[abc452_b](https://atcoder.jp/contests/abc452/tasks/abc452_b)

给定 $H\times W$ 网格，边框涂黑输出 `#`，内部涂白输出 `.`，打印最终图案。

**解题思路**

枚举每个格子 $(i,j)$：若 `i==1 || i==H || j==1 || j==W` 输出 `#`，否则输出 `.`。

**代码**
```cpp
void solve() {
    int h, w;
    cin >> h >> w;
    for (int i = 1; i <= h; i++) {
        for (int j = 1; j <= w; j++) {
            cout << ((i == 1 || i == h || j == 1 || j == w) ? '#' : '.');
        }
        cout << '\n';
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(HW)$。空间复杂度：$O(1)$。

**变形**

1. 多层边框可按“到边界最短距离”分类。
2. 障碍版边框可转为构造/模拟。

## C - Fishbones

**题意**

题目链接：[abc452_c](https://atcoder.jp/contests/abc452/tasks/abc452_c)

有一根主骨和 $N$ 根肋骨。主骨字符串长度必须是 $N$。第 $i$ 根肋骨要求：长度为 $A_i$，且第 $B_i$ 个字符等于主骨第 $i$ 个字符。所有字符串都从给定的 $M$ 个字符串中选取，允许重复。对每个 $S_j$，判断能否作为主骨字符串。

**解题思路**

定义布尔状态 $ok[\ell][p][c]$ 表示：是否存在某个给定字符串，满足长度为 $\ell$ 且第 $p$ 位字符为字符 $c$。

预处理所有字符串后，检查候选主骨串 `t`：

1. 若 $|t| \ne N$，直接输出 `No`。
2. 对每个 $i$，检查 $ok[A_{i}][B_{i}][t_{i}]$ 是否成立，其中 $t_{i}$ 表示主骨串第 $i$ 个字符。
3. 全满足则 `Yes`。

关键是“允许重复使用字符串”，各肋骨选择彼此独立。

**代码**
```cpp
void solve() {
    int n;
    cin >> n;
    vector<int> a(n + 1), b(n + 1);
    for (int i = 1; i <= n; i++) cin >> a[i] >> b[i];

    int m;
    cin >> m;
    vector<string> s(m);
    for (int i = 0; i < m; i++) cin >> s[i];

    vector<vector<vector<int>>> ok(11, vector<vector<int>>(11, vector<int>(26, 0)));
    for (const string &t : s) {
        int len = (int)t.size();
        for (int i = 0; i < len; i++) {
            ok[len][i + 1][t[i] - 'a'] = 1;
        }
    }

    for (const string &t : s) {
        if ((int)t.size() != n) {
            cout << "No\n";
            continue;
        }
        bool good = true;
        for (int i = 1; i <= n; i++) {
            if (!ok[a[i]][b[i]][t[i - 1] - 'a']) {
                good = false;
                break;
            }
        }
        cout << (good ? "Yes\n" : "No\n");
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

**复杂度分析**

设所有字符串总长度为 $L$，时间复杂度：$O(L + MN)$，其中 $N\le 10$。空间复杂度：$O(L)$（存输入）+ 常数状态表。

**变形**

1. 若不允许重复使用字符串，会转为匹配/状压问题。
2. 若要求方案数，可做计数版 DP 或匹配计数。

## D - No-Subsequence Substring

**题意**

题目链接：[abc452_d](https://atcoder.jp/contests/abc452/tasks/abc452_d)

给定字符串 $S,T$，统计 $S$ 的非空子串中，有多少个不包含 $T$ 作为子序列。

**解题思路**

这题的核心不是直接数合法子串，而是对每个左端点找一个“分界点”。

设 $R_l$ 是最早满足 $S[l..R_l]$ 含有 $T$ 作为子序列的右端点。

1. 若 $R_l$ 不存在，说明从 $l$ 出发的所有子串都不含 $T$，贡献是 $n-l$。
2. 若 $R_l$ 存在，则右端点在区间 $[l,R_l-1]$ 时都合法，贡献是 $R_l-l$。

为什么这个划分成立：

1. 一旦某个 $S[l..r]$ 已经包含 $T$，再把右端点向右扩展，原先的匹配位置仍然存在，所以依然包含。
2. 因此对固定 $l$，合法性一定是“前缀合法、后缀不合法”的单调结构，关键只剩找到最早失效点。

接下来要快速计算每个 $R_l$。定义 $ne_{i,c}$ 为“从位置 $i$ 往后，字符 $c$ 第一次出现的位置”，若不存在记为 $n$。从右向左预处理：

1. 初始化 $ne_{n,c}=n$。
2. 对 $i=n-1$ 到 $0$，先继承下一行，再把当前字符位置更新到对应列。

有了 $ne$ 之后，对每个左端点 $l$ 贪心匹配 $T$：

1. 指针 $p=l$。
2. 依次处理 $T$ 的字符 $t_k$，跳到 $p\leftarrow ne_{p,t_k}+1$。
3. 某步若 $ne_{p,t_k}=n$，则该 $l$ 匹配失败，贡献 $n-l$。
4. 全部匹配成功时，最早右端点是 $R_l=p-1$，贡献 $R_l-l$。

为什么贪心正确：每一步都取当前字符最靠左的位置，不会损害后续可行性，且会最小化最后落点，因此得到的正是最早右端点。

**代码**
```cpp
void solve() {
    string s, t;
    cin >> s >> t;
    int n = (int)s.size();

    vector<array<int, 26>> ne(n + 1);
    for (int c = 0; c < 26; c++) ne[n][c] = n;
    for (int i = n - 1; i >= 0; i--) {
        ne[i] = ne[i + 1];
        ne[i][s[i] - 'a'] = i;
    }

    long long ans = 0;
    for (int l = 0; l < n; l++) {
        int p = l;
        bool ok = true;
        for (char ch : t) {
            if (p >= n || ne[p][ch - 'a'] == n) {
                ok = false;
                break;
            }
            p = ne[p][ch - 'a'] + 1;
        }
        if (!ok) ans += n - l;
        else ans += (p - 1) - l;
    }

    cout << ans << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(26|S| + |S||T|)$，且 $|T|\le 50$。空间复杂度：$O(26|S|)$。

**变形**

1. 统计“包含 $T$ 作为子序列”的子串数，可用总子串数减法。
2. 多次询问不同左端点时可复用同一 `next` 表。

## E - You WILL Like Sigma Problem

**题意**

题目链接：[abc452_e](https://atcoder.jp/contests/abc452/tasks/abc452_e)

给定正整数数组 $A,B$，求
$$
\sum_{i=1}^{N}\sum_{j=1}^{M} A_iB_j(i\bmod j)
$$
对 $998244353$ 取模。

**解题思路**

先做公式拆分。利用恒等式

$$
i\bmod j=i-j\left\lfloor\frac{i}{j}\right\rfloor
$$

原式变为：

$$
\text{Ans}=\left(\sum_{i=1}^N A_i i\right)\left(\sum_{j=1}^M B_j\right)-\sum_{j=1}^M (jB_j)\sum_{i=1}^N A_i\left\lfloor\frac{i}{j}\right\rfloor
$$

第一项可以直接 $O(N+M)$ 算出。难点在第二项，设

$$
F(j)=\sum_{i=1}^{N}A_i\left\lfloor\frac{i}{j}\right\rfloor
$$

则第二项是 $\sum_{j=1}^{M}(jB_j)F(j)$。

固定一个 $j$ 时，$\left\lfloor\frac{i}{j}\right\rfloor$ 在每段区间上是常数：

$$
i\in[kj,(k+1)j-1]\Rightarrow\left\lfloor\frac{i}{j}\right\rfloor=k
$$

因此

$$
F(j)=\sum_{k=1}^{\lfloor N/j\rfloor}k\cdot\sum_{i=kj}^{\min(N,(k+1)j-1)}A_i
$$

再配合前缀和 $pre_x=\sum_{t=1}^{x}A_t$，即可 $O(1)$ 求任意段和，固定 $j$ 的复杂度是 $O(N/j)$。

总复杂度为

$$
O\left(N+M+\sum_{j=1}^{\min(N,M)}\frac{N}{j}\right)=O(N\log N+M)
$$

这样就能通过 $N,M\le 5\times 10^5$。

**代码**
```cpp
void solve() {
    const long long MOD = 998244353;

    int n, m;
    cin >> n >> m;
    vector<long long> a(n + 1), b(m + 1), pre(n + 1, 0);

    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        pre[i] = (pre[i - 1] + a[i]) % MOD;
    }
    for (int j = 1; j <= m; j++) cin >> b[j];

    long long sa = 0, sb = 0;
    for (int i = 1; i <= n; i++) sa = (sa + (a[i] % MOD) * i) % MOD;
    for (int j = 1; j <= m; j++) sb = (sb + b[j]) % MOD;

    long long ans = sa * sb % MOD;

    int lim = min(n, m);
    for (int j = 1; j <= lim; j++) {
        long long s = 0;
        for (int l = j, k = 1; l <= n; l += j, k++) {
            int r = min(n, l + j - 1);
            long long seg = (pre[r] - pre[l - 1] + MOD) % MOD;
            s = (s + seg * k) % MOD;
        }
        long long dec = (b[j] % MOD) * j % MOD * s % MOD;
        ans = (ans - dec + MOD) % MOD;
    }

    cout << ans << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

**复杂度分析**

时间复杂度：$O\!\left(N+M+\sum_{j=1}^{\min(N,M)}\frac{N}{j}\right)=O(N\log N+M)$。空间复杂度：$O(N+M)$。

**变形**

1. 若把 $i\bmod j$ 改成 $\left\lfloor i/j\right\rfloor$，后半部分可直接复用。
2. 多组不同 $B$ 查询时可先预处理每个 $j$ 对应贡献。

## F - Interval Inversion Count

**题意**

题目链接：[abc452_f](https://atcoder.jp/contests/abc452/tasks/abc452_f)

给定排列 $P$ 和整数 $K$，统计区间 $[l,r]$ 中逆序对数恰好为 $K$ 的区间个数。

**解题思路**

直接数“逆序对恰好等于 $K$”很难，先改成前缀计数。

定义 $cal(X)$ 为“逆序对数 $\le X$ 的区间个数”，那么答案是：
$$
\text{cal}(K)-\text{cal}(K-1)
$$

接下来只需实现 $cal(X)$。用双指针维护窗口 $[l,r]$，以及窗口逆序对数 $inv$。当右端点加入新元素后，若 $inv>X$ 就持续右移左端点，直到恢复 $inv\le X$。此时以 $r$ 结尾的合法区间个数是 $r-l+1$。

关键是动态维护 $inv$，用树状数组维护窗口内各值出现次数。

1. 右端加入 $y=P_r$：新增逆序对是窗口内大于 $y$ 的元素数。
2. 左端删除 $z=P_l$：减少逆序对是窗口内小于 $z$ 的元素数。

对应公式：

$$
\Delta_{\text{add}}=(r-l)-\#\{x\in[l,r-1]\mid P_x\le y\}
$$

$$
\Delta_{\text{remove}}=\#\{x\in[l+1,r]\mid P_x<z\}
$$

因为原数组是 $1..N$ 的排列，值域正好可直接作为树状数组下标，单次更新/查询都是 $O(\log N)$，且每个指针只单调移动一次，总复杂度 $O(N\log N)$。

为什么双指针正确：固定右端点时，左端点越右，区间只会删元素，逆序对不会增加，所以合法左端点形成连续后缀。

**代码**
```cpp
struct BIT {
    int n;
    vector<int> tr;
    BIT() {}
    BIT(int n_) : n(n_), tr(n_ + 1, 0) {}

    void add(int x, int v) {
        while (x <= n) {
            tr[x] += v;
            x += x & -x;
        }
    }

    int sum(int x) {
        int r = 0;
        while (x > 0) {
            r += tr[x];
            x -= x & -x;
        }
        return r;
    }
};

long long cal(const vector<int> &p, long long x) {
    if (x < 0) return 0;
    int n = (int)p.size() - 1;

    BIT fw(n);
    long long ans = 0, inv = 0;
    int l = 1;

    for (int r = 1; r <= n; r++) {
        int y = p[r];
        int len = r - l;
        int le = fw.sum(y);
        inv += len - le;
        fw.add(y, 1);

        while (inv > x) {
            int z = p[l];
            inv -= fw.sum(z - 1);
            fw.add(z, -1);
            l++;
        }
        ans += r - l + 1;
    }

    return ans;
}

void solve() {
    int n;
    long long k;
    cin >> n >> k;
    vector<int> p(n + 1);
    for (int i = 1; i <= n; i++) cin >> p[i];

    cout << cal(p, k) - cal(p, k - 1) << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(N\log N)$。空间复杂度：$O(N)$。

**变形**

1. 若只求逆序对数 $\le K$ 的区间数，直接输出 $cal(K)$。
2. 对非排列数组也可做，注意重复值计数细节。

## G - 221 Substring

**题意**

题目链接：[abc452_g](https://atcoder.jp/contests/abc452/tasks/abc452_g)

定义序列是 221 序列：其每个连续相等段的“段长 = 段值”。给定 $A_1..A_N$（$A_i\in[1,9]$），求作为其非空连续子序列出现过的“不同 221 序列”数量。

**解题思路**

先把原数组做 RLE，得到段值序列 $c_1,c_2,\dots,c_m$ 和段长序列 $d_1,d_2,\dots,d_m$。

一个 221 序列的每段长度由段值唯一决定，所以只要确定 run 值子串，展开后的真实序列也唯一。于是问题转成：统计 run 值串里“可实现为 221”的不同子串个数。

对一个 run 值子串 $c_l..c_r$，充要条件是：

1. 两端段可截取，要求 $d_l\ge c_l$ 且 $d_r\ge c_r$。
2. 中间段必须完整经过，要求对 $l<i<r$ 有 $d_i=c_i$。

记

$$
ok_i=[d_i\ge c_i],\quad eq_i=[d_i=c_i]
$$

固定右端点 $r$ 后，合法左端点形成一个连续区间 $[s_r,r]$，于是合法长度恰好是 $1..L_r$，其中 $L_r=r-s_r+1$。

求 $s_r$ 的方法：维护前缀最后一个不满足 $eq$ 的位置。设该位置为 $q$，若 $q=0$ 则 $s_r=1$；否则根据 $ok_q$ 决定 $s_r=q$ 或 $q+1$。

这样，我们得到每个结尾位置 $r$ 的“允许最大长度” $L_r$。剩下就是一个带长度上界的不同子串计数问题，使用后缀自动机。

1. 在 run 值串上建 SAM。
2. 每个前缀结尾 $r$ 对应一个终止状态，把该状态的 $mx$ 用 $L_r$ 更新。
3. 按状态长度从大到小沿 suffix link 传播 $mx$，得到每个状态可支持的最大合法长度。

SAM 状态 $v$ 表示长度区间

$$
(\mathrm{len}[link(v)],\ \mathrm{len}[v]]
$$

其中合法长度上界是 $\min(\mathrm{len}[v],mx[v])$，所以贡献为

$$
\max\bigl(0,\min(\mathrm{len}[v],mx[v]) - \mathrm{len}[\mathrm{link}[v]]\bigr)
$$

所有状态贡献求和即答案。整体是线性复杂度：RLE、建 SAM、计数排序拓扑、一次传播都为 $O(N)$。

**代码**
```cpp
struct SAM {
    int sz = 1, last = 1;
    vector<array<int, 10>> nx;
    vector<int> link, len, mx;

    SAM(int n) {
        nx.assign(2 * n + 5, {});
        link.assign(2 * n + 5, 0);
        len.assign(2 * n + 5, 0);
        mx.assign(2 * n + 5, 0);
    }

    void extend(int x) {
        int cur = ++sz;
        len[cur] = len[last] + 1;

        int p = last;
        while (p && !nx[p][x]) {
            nx[p][x] = cur;
            p = link[p];
        }

        if (!p) {
            link[cur] = 1;
        } else {
            int q = nx[p][x];
            if (len[q] == len[p] + 1) {
                link[cur] = q;
            } else {
                int cl = ++sz;
                nx[cl] = nx[q];
                link[cl] = link[q];
                len[cl] = len[p] + 1;
                while (p && nx[p][x] == q) {
                    nx[p][x] = cl;
                    p = link[p];
                }
                link[q] = link[cur] = cl;
            }
        }
        last = cur;
    }
};

void solve() {
    int n;
    cin >> n;
    vector<int> a(n + 1);
    for (int i = 1; i <= n; i++) cin >> a[i];

    vector<int> c(1), d(1);
    for (int i = 1; i <= n; i++) {
        if (i == 1 || a[i] != a[i - 1]) {
            c.push_back(a[i]);
            d.push_back(1);
        } else {
            d.back()++;
        }
    }

    int m = (int)c.size() - 1;
    vector<int> ok(m + 1), eq(m + 1), preBad(m + 1, 0), L(m + 1, 0);

    for (int i = 1; i <= m; i++) {
        ok[i] = (d[i] >= c[i]);
        eq[i] = (d[i] == c[i]);
        preBad[i] = preBad[i - 1];
        if (!eq[i]) preBad[i] = i;
    }

    for (int r = 1; r <= m; r++) {
        if (!ok[r]) continue;
        int q = preBad[r - 1];
        int s;
        if (q == 0) s = 1;
        else s = ok[q] ? q : q + 1;
        L[r] = r - s + 1;
    }

    SAM sam(m);
    vector<int> ed(m + 1, 0);
    for (int i = 1; i <= m; i++) {
        sam.extend(c[i]);
        ed[i] = sam.last;
        sam.mx[ed[i]] = max(sam.mx[ed[i]], L[i]);
    }

    vector<int> cnt(m + 1, 0), ord(sam.sz + 1, 0);
    for (int v = 1; v <= sam.sz; v++) cnt[sam.len[v]]++;
    for (int i = 1; i <= m; i++) cnt[i] += cnt[i - 1];
    for (int v = 1; v <= sam.sz; v++) ord[cnt[sam.len[v]]--] = v;

    for (int i = sam.sz; i >= 2; i--) {
        int v = ord[i];
        sam.mx[sam.link[v]] = max(sam.mx[sam.link[v]], sam.mx[v]);
    }

    long long ans = 0;
    for (int v = 2; v <= sam.sz; v++) {
        int up = min(sam.len[v], sam.mx[v]);
        int lo = sam.len[sam.link[v]];
        if (up > lo) ans += up - lo;
    }

    cout << ans << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

**复杂度分析**

时间复杂度：$O(N)$。空间复杂度：$O(N)$。

**变形**

1. 若统计出现次数而非不同序列，可直接累加各右端点的 $L_r$。
2. 若值域变大，可离散化或改为 map 转移版 SAM。
