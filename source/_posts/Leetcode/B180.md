## 3894 - Traffic Signal Color

**题意**

题目链接：[3894. Traffic Signal Color](https://leetcode.com/problems/traffic-signal-color/)

给定整数 `timer`，按下列规则返回当前信号灯状态：

1. `timer == 0` 返回 `"Green"`。
2. `timer == 30` 返回 `"Orange"`。
3. `30 < timer <= 90` 返回 `"Red"`。
4. 其余返回 `"Invalid"`。

**解题思路**

直接按题意做条件分支判断即可。

**代码**

```cpp
class Solution {
public:
    string trafficSignal(int timer) {
        if (timer == 0) return "Green";
        if (timer == 30) return "Orange";
        if (30 < timer && timer <= 90) return "Red";
        return "Invalid";
    }
};
```

**复杂度分析**

时间复杂度：$O(1)$。空间复杂度：$O(1)$。

## 3895 - Count Digit Appearances

**题意**

题目链接：[3895. Count Digit Appearances](https://leetcode.com/problems/count-digit-appearances/)

给定数组 `nums` 和数字 `digit`，统计 `digit` 在所有元素十进制表示中出现的总次数。

**解题思路**

对每个数 `x` 做十进制拆位：

1. 取当前最低位 `x % 10`，若等于 `digit` 则计数加一。
2. 令 `x /= 10`，继续处理下一位。
3. 直到 `x == 0`。

将所有元素的贡献累加即为答案。

**代码**

```cpp
class Solution {
public:
    int countDigitOccurrences(vector<int>& nums, int digit) {
        int ans = 0;
        for (int x : nums) {
            while (x > 0) {
                if (x % 10 == digit) {
                    ++ans;
                }
                x /= 10;
            }
        }
        return ans;
    }
};
```

**复杂度分析**

设单个数的十进制位数上界为 $k$，数组长度为 $n$。

时间复杂度：$O(nk)$。空间复杂度：$O(1)$。

## 3896 - Minimum Operations to Transform Array into Alternating Prime

**题意**

题目链接：[3896. Minimum Operations to Transform Array into Alternating Prime](https://leetcode.com/problems/minimum-operations-to-transform-array-into-alternating-prime/)

给定数组 `nums`，每次操作可将任意元素加一。目标是使：

1. 偶数下标元素为质数。
2. 奇数下标元素为非质数。

求最小操作次数。

**解题思路**

每个位置的修改互不影响，可以逐位独立计算最小代价后求和。

1. 奇数下标要求非质数：
   若当前已非质数，代价为 `0`；
   若当前是质数：
   `x == 2` 时代价为 `2`（`2 -> 4`）；
   其余质数代价为 `1`（奇质数加一变为大于 `2` 的偶数，必为非质数）。
2. 偶数下标要求质数：
   需要找到不小于 `x` 的最小质数 `nextPrime[x]`，代价为 `nextPrime[x] - x`。

为快速求 `nextPrime`，先筛出 `isPrime`，再从右向左预处理最近质数位置。由于 $x \le 10^5$，取上界 `2 * max(nums)` 即可保证存在后继质数。

**代码**

```cpp
class Solution {
public:
    int minOperations(vector<int>& nums) {
        int mx = 0;
        for (int x : nums) mx = max(mx, x);
        int lim = max(10, 2 * mx + 5);

        vector<int> isPrime(lim + 1, 1);
        isPrime[0] = isPrime[1] = 0;
        for (int i = 2; 1LL * i * i <= lim; ++i) {
            if (!isPrime[i]) continue;
            for (int j = i * i; j <= lim; j += i) {
                isPrime[j] = 0;
            }
        }

        vector<int> nextPrime(lim + 2, -1);
        int last = -1;
        for (int i = lim; i >= 0; --i) {
            if (isPrime[i]) last = i;
            nextPrime[i] = last;
        }

        long long ans = 0;
        for (int i = 0; i < (int)nums.size(); ++i) {
            int x = nums[i];
            if (i % 2 == 0) {
                ans += nextPrime[x] - x;
            } else {
                if (isPrime[x]) ans += (x == 2 ? 2 : 1);
            }
        }
        return (int)ans;
    }
};
```

**复杂度分析**

设 $M = \max(nums)$，$n = nums.length$。

时间复杂度：$O(M \log\log M + n)$。空间复杂度：$O(M)$。

**变形**

若允许 `+1/-1` 两种操作，则每个位置需要比较“左右最近合法值”的距离，不能只看右侧。

## 3897 - Maximum Value of Concatenated Binary Segments

**题意**

题目链接：[3897. Maximum Value of Concatenated Binary Segments](https://leetcode.com/problems/maximum-value-of-concatenated-binary-segments/)

给定两数组 `nums1`、`nums0`。第 $i$ 段由 `nums1[i]` 个 `'1'` 后接 `nums0[i]` 个 `'0'` 组成。可任意重排各段后拼接成总二进制串。求该二进制串表示整数的最大值（对 $10^9+7$ 取模）。

**解题思路**

按拼接最大值的通用做法直接处理即可。

1. 先把每一段构造成字符串 $s_i = 1^{a_i}0^{b_i}$。
2. 排序比较器使用 `A+B > B+A`（字典序），若成立则 `A` 排在 `B` 前。
3. 排好序后依次拼接。由于总串可能很长，不直接转十进制，而是边扫描边按二进制更新模值：`ans = (ans * 2 + bit) % MOD`。

**代码**

```cpp
class Solution {
public:
    int maxValue(vector<int>& nums1, vector<int>& nums0) {
        const int MOD = 1000000007;
        int n = (int)nums1.size();
        vector<string> v;
        v.reserve(n);

        for (int i = 0; i < n; ++i) {
            string s(nums1[i], '1');
            s += string(nums0[i], '0');
            v.push_back(s);
        }

        sort(v.begin(), v.end(), [&](const string& a, const string& b) {
            return a + b > b + a;
        });

        long long ans = 0;
        for (const string& s : v) {
            for (char c : s) {
                ans = (ans * 2 + (c - '0')) % MOD;
            }
        }

        return (int)ans;
    }
};
```

**复杂度分析**

设段数为 $n$，总长度为 $S=\sum_i(\text{nums1}_i+\text{nums0}_i)$。

构造与扫描总串代价为 $O(S)$。排序阶段使用比较器 `A+B > B+A`，单次比较最坏为 $O(|A|+|B|)$，总体可写为 $O(n\log n\cdot L)$，其中 $L$ 是平均比较长度。

整体时间复杂度：$O(S + n\log n\cdot L)$。空间复杂度：$O(S)$。

**另一种写法"**
直接对nums排序, 不构造字符串
```cpp
#include <bits/stdc++.h>
using namespace std;

#ifdef LOCAL
#include "debug.h"
#else
#define debug(...) 42
#endif
using LL = long long;
using ULL = unsigned long long;

const int MOD = 1e9 + 7;

class Solution {
public:
    int maxValue(vector<int>& nums1, vector<int>& nums0) {
        int n = nums1.size();
        vector<pair<int, int>> v;
        v.reserve(n);

        for (int i = 0; i < n; i++) {
            v.push_back({nums1[i], nums0[i]});
        }

        sort(v.begin(), v.end(), [&](pair<int, int> x, pair<int, int> y) {
            return make_tuple(x.second == 0, x.first, -x.second) >
                   make_tuple(y.second == 0, y.first, -y.second);
        });

        LL ans = 0;
        for (auto [a, b] : v) {
            for (int i = 0; i < a; i++) {
                ans = (ans * 2 + 1) % MOD;
            }
            for (int i = 0; i < b; i++) {
                ans = (ans * 2) % MOD;
            }
        }

        return (int)ans;
    }
};
```

**变形**

若每段改为任意二进制串，可直接使用通用比较器 `A+B > B+A` 进行排序。
