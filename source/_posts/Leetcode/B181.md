## A - Valid Digit Number

**题意**

题目链接：[3908. Valid Digit Number](https://leetcode.com/problems/valid-digit-number/description/)

给定整数 $n$ 和数字 $x$，判断 $n$ 是否满足：
- 十进制表示中至少出现一次 $x$；
- 十进制表示的第一位不是 $x$。

$0 \le n \le 10^5$，$0 \le x \le 9$。

**解题思路**

将整数 $n$ 转为字符串 $s$，记 $c = x + '0'$。直接判断两个条件：
1. $s[0] \neq c$；
2. $s$ 中存在字符 $c$。

二者同时满足返回 `true`。

**代码**

```cpp
class Solution {
public:
    bool validDigit(int n, int x) {
        string s = to_string(n);
        char c = char('0' + x);

        if (s[0] == c) {
            return false;
        }

        for (char ch : s) {
            if (ch == c) {
                return true;
            }
        }

        return false;
    }
};
```

**复杂度分析**

设 $n$ 的位数为 $d$，时间复杂度 $O(d)$，空间复杂度 $O(d)$。

## B - Compare Sums of Bitonic Parts

**题意**

题目链接：[3909. Compare Sums of Bitonic Parts](https://leetcode.com/contest/biweekly-contest-181/problems/compare-sums-of-bitonic-parts/submissions/1988040247/)

给定长度为 $n$ 的双调数组 `nums`（严格递增到峰值，再严格递减），峰值下标为 $p$。定义：
- 上升部分和：$nums[0] + \dots + nums[p]$；
- 下降部分和：$nums[p] + \dots + nums[n-1]$。

返回 $0$ 表示上升部分和更大，$1$ 表示下降部分和更大，$-1$ 表示二者相等。

$3 \le n \le 10^5$，$1 \le nums[i] \le 10^9$。

**解题思路**

双调数组中，峰值是第一个满足 $nums[i] > nums[i+1]$ 的位置 $i$，可一次扫描找到。

分别求两个区间的和并比较即可。由于 $nums[i]$ 最大 $10^9$，区间和可能超过 `int` 范围，需用 `long long`。

**代码**

```cpp
using LL = long long;
using ULL = unsigned long long;

class Solution {
public:
    int compareBitonicSums(vector<int>& nums) {
        int n = nums.size();
        int p = 0;

        for (int i = 0; i + 1 < n; i++) {
            if (nums[i] > nums[i + 1]) {
                p = i;
                break;
            }
        }

        LL a = 0, b = 0;

        for (int i = 0; i <= p; i++) {
            a += nums[i];
        }

        for (int i = p; i < n; i++) {
            b += nums[i];
        }

        if (a > b) return 0;
        if (a < b) return 1;
        return -1;
    }
};
```

**复杂度分析**

时间复杂度 $O(n)$，空间复杂度 $O(1)$（不计输入存储）。

## C - Count Connected Subgraphs with Even Node Sum

**题意**

题目链接：[3910. Count Connected Subgraphs with Even Node Sum](https://leetcode.com/problems/count-connected-subgraphs-with-even-node-sum/description/)

给定 $n$ 个节点的无向图，节点权值 $nums[i] \in \{0,1\}$。统计有多少个非空节点集合 $s$ 满足：
1. $s$ 的诱导子图连通；
2. $s$ 中节点权值之和为偶数。

$1 \le n \le 13$。

**解题思路**

$n \le 13$ 很小，直接用状态压缩枚举所有非空子集。

对于每个子集 $msk$：
- 用异或判断权值和的奇偶性（$nums[i]$ 为 $0/1$，奇偶性等于异或和）；
- 用 BFS/DFS 在诱导子图内从任一节点出发，判断能否访问完所有节点。

**代码**

```cpp
using LL = long long;
using ULL = unsigned long long;

class Solution {
public:
    int evenSumSubgraphs(vector<int>& nums, vector<vector<int>>& edges) {
        int n = nums.size();

        vector<int> adj(n);
        for (auto &e : edges) {
            int u = e[0], v = e[1];
            adj[u] |= 1 << v;
            adj[v] |= 1 << u;
        }

        int ans = 0;

        for (int msk = 1; msk < (1 << n); msk++) {
            int p = 0;

            for (int i = 0; i < n; i++) {
                if (msk >> i & 1) {
                    p ^= nums[i];
                }
            }

            if (p) continue;

            int st = __builtin_ctz(msk);
            int vis = 0;
            int que = 1 << st;

            while (que) {
                int u = __builtin_ctz(que);
                que ^= 1 << u;

                if (vis >> u & 1) continue;

                vis |= 1 << u;

                int nxt = adj[u] & msk & ~vis;
                que |= nxt;
            }

            if (vis == msk) {
                ans++;
            }
        }

        return ans;
    }
};
```

**复杂度分析**

子集总数 $2^n - 1$，对每个子集 $O(n)$ 判断，总时间 $O(n \cdot 2^n)$，空间 $O(n)$。

## D - K-th Smallest Remaining Even Integer in Subarray Queries

**题意**

题目链接：[3911. K-th Smallest Remaining Even Integer in Subarray Queries](https://leetcode.com/problems/k-th-smallest-remaining-even-integer-in-subarray-queries/description/)

给定严格递增数组 `nums` 和 $m$ 个查询，每个查询 $[l, r, k]$：
- 在正偶数序列 $2,4,6,\dots$ 中删除 `nums[l..r]` 中出现的偶数；
- 返回剩余序列中第 $k$ 小的数。

$1 \le n, m \le 10^5$，$1 \le k \le 10^9$。

**解题思路**

只需关心 `nums[l..r]` 中的偶数。将所有正偶数除以 $2$，问题转化为：
- 从正整数序列中删除若干数，求第 $k$ 小剩余正整数。

设删除的正整数集合为 $val[a..b]$。对任意 $x$，$1..x$ 中剩余数量为 $x - \text{cnt}(val[a..b] \le x)$，随 $x$ 单调不减，可二分求解。

**代码**

```cpp
#include <bits/stdc++.h>
using namespace std;

#ifdef LOCAL
#include "debug.h"
#else
#define debug(...) 42
#endif
using LL = long long;
using ULL = unsigned long long;

class Solution {
public:
    vector<int> kthRemainingInteger(vector<int>& nums, vector<vector<int>>& queries) {
        int n = nums.size();

        vector<int> pos;
        vector<int> val;

        for (int i = 0; i < n; i++) {
            if (nums[i] % 2 == 0) {
                pos.push_back(i);
                val.push_back(nums[i] / 2);
            }
        }

        vector<int> ans;

        for (auto &q : queries) {
            int l = q[0], r = q[1];
            LL k = q[2];

            int a = lower_bound(pos.begin(), pos.end(), l) - pos.begin();
            int b = upper_bound(pos.begin(), pos.end(), r) - pos.begin();

            LL lo = 1;
            LL hi = k + (b - a);

            while (lo < hi) {
                LL mid = (lo + hi) / 2;

                int del = upper_bound(val.begin() + a, val.begin() + b, mid) - (val.begin() + a);
                LL rem = mid - del;

                if (rem >= k) {
                    hi = mid;
                } else {
                    lo = mid + 1;
                }
            }

            ans.push_back(lo * 2);
        }

        return ans;
    }
};
```

**复杂度分析**

预处理 $O(n)$。每个查询 $O(\log 10^9 \cdot \log n)$，总时间 $O(n + m \log 10^9 \log n)$，空间 $O(n)$。
