## 3898 - Find the Degree of Each Vertex

**题意**

题目链接：[3898. Find the Degree of Each Vertex](https://leetcode.com/problems/find-the-degree-of-each-vertex/)

给定无向图的邻接矩阵 `matrix`。其中 `matrix[i][j]=1` 表示顶点 `i` 与顶点 `j` 有边，`matrix[i][j]=0` 表示无边。返回每个顶点的度数。

**解题思路**

无向图的邻接矩阵中，第 `i` 行的 `1` 个数就是顶点 `i` 的度数。逐行求和即可。

**代码**

```cpp
class Solution {
public:
    vector<int> findDegrees(vector<vector<int>>& matrix) {
        int n = (int)matrix.size();
        vector<int> ans(n, 0);
        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < n; ++j) {
                ans[i] += matrix[i][j];
            }
        }
        return ans;
    }
};
```

**复杂度分析**

时间复杂度：$O(n^2)$。空间复杂度：$O(n)$。

## 3899 - Angles of a Triangle

**题意**

题目链接：[3899. Angles of a Triangle](https://leetcode.com/problems/angles-of-a-triangle/)

给定三条边长 `sides`。若它们不能构成面积为正的三角形，则返回空数组；否则返回三个内角（单位为度），按非降序排列。

**解题思路**

先排序得到 $a\le b\le c$，用三角形不等式判断 `a + b > c`。若不成立，直接返回空数组。

若成立，使用余弦定理计算三个角：

$$
\begin{aligned}
\cos A &= \frac{b^2 + c^2 - a^2}{2bc},\\
\cos B &= \frac{a^2 + c^2 - b^2}{2ac},\\
\cos C &= \frac{a^2 + b^2 - c^2}{2ab}.
\end{aligned}
$$

再通过 `acos` 转为角度。为避免浮点误差导致余弦值越界，先将余弦值截断到 $[-1,1]$。

**代码**

```cpp
class Solution {
public:
    double get(double x) {
        x = max(-1.0, min(1.0, x));
        const double pi = acos(-1.0);
        return acos(x) * 180.0 / pi;
    }

    vector<double> internalAngles(vector<int>& sides) {
        vector<double> s(3);
        for (int i = 0; i < 3; i++) s[i] = sides[i];
        sort(s.begin(), s.end());

        double a = s[0], b = s[1], c = s[2];
        if (a + b <= c) return {};

        double ca = (b * b + c * c - a * a) / (2.0 * b * c);
        double cb = (a * a + c * c - b * b) / (2.0 * a * c);
        double cc = (a * a + b * b - c * c) / (2.0 * a * b);

        vector<double> ans;
        ans.push_back(get(ca));
        ans.push_back(get(cb));
        ans.push_back(get(cc));

        sort(ans.begin(), ans.end());
        return ans;
    }
};
```

**复杂度分析**

时间复杂度：$O(1)$。空间复杂度：$O(1)$。

## 3900 - Longest Balanced Substring After One Swap

**题意**

题目链接：[3900. Longest Balanced Substring After One Swap](https://leetcode.com/problems/longest-balanced-substring-after-one-swap/)

给定二进制串 `s`。最多允许交换一次任意两个位置，然后选择一个子串。若子串中 `0` 与 `1` 个数相同，则称为平衡子串。求可得到的最长平衡子串长度。

**解题思路**

设 `1 -> +1`、`0 -> -1`，则子串和为 $\#1-\#0$。平衡子串等价于子串和为 `0`。

对一个固定子串，进行一次交换时：

1. 若交换点都在子串内或都在子串外，子串和不变。
2. 若恰有一个交换点在子串内，子串内 `1` 的个数最多变化 `1`，因此子串和最多变化 `2`。

所以可被一次交换修正为平衡的候选子串，其原始和只可能是 `0`、`2`、`-2`。

进一步讨论可得：

1. 和为 `0`：直接合法。
2. 和为 `2`：需从子串外换入一个 `0`，可推出长度约束 $L\le 2c_0$（`c0` 为全串 `0` 总数）。
3. 和为 `-2`：需从子串外换入一个 `1`，可推出长度约束 $L\le 2c_1$（`c1` 为全串 `1` 总数）。

问题转化为求三类“定和且带长度上界”的最长子数组：

1. `sum = 0, maxLen = n`
2. `sum = 2, maxLen = 2*c0`
3. `sum = -2, maxLen = 2*c1`

设前缀和为 `pre`，子数组和为 `t` 等价于 `pre[r]-pre[l]=t`。把所有前缀位置按前缀和值分桶。对一组桶值 `(x, x+t)`，右端点在后者，左端点在前者，用双指针维护满足 `r-l<=maxLen` 的最早合法 `l`，即可得到该组最大长度。三种 `t` 取最大值即答案。

**代码**

```cpp
class Solution {
public:
    // 求和为 t 且长度不超过 m 的最长子数组长度
    int cal(vector<vector<int>>& pos, int n, int t, int m) {
        if (m <= 0) return 0;
        m = min(m, n);

        int ans = 0;
        int sz = 2 * n + 1;

        for (int i = 0; i < sz; i++) {
            int j = i + t;
            if (j < 0 || j >= sz) continue;

            auto& a = pos[i];
            auto& b = pos[j];
            if (a.empty() || b.empty()) continue;

            int p = 0;
            for (int x : b) {
                while (p < (int)a.size() && a[p] < x - m) p++;
                if (p < (int)a.size() && a[p] < x) {
                    ans = max(ans, x - a[p]);
                }
            }
        }
        return ans;
    }

    int longestBalanced(string s) {
        int n = s.size();
        int c0 = 0, c1 = 0;

        vector<int> pre(n + 1, 0);
        for (int i = 0; i < n; i++) {
            if (s[i] == '0') {
                c0++;
                pre[i + 1] = pre[i] - 1;
            } else {
                c1++;
                pre[i + 1] = pre[i] + 1;
            }
        }

        int offset = n;
        vector<vector<int>> pos(2 * n + 1);
        for (int i = 0; i <= n; i++) {
            pos[pre[i] + offset].push_back(i);
        }

        int ans = 0;
        ans = max(ans, cal(pos, n, 0, n));
        ans = max(ans, cal(pos, n, 2, 2 * c0));
        ans = max(ans, cal(pos, n, -2, 2 * c1));
        return ans;
    }
};
```

**复杂度分析**

设串长为 $n$。构造前缀和与分桶为 $O(n)$。每次 `cal` 中，所有桶内位置总数为 $n+1$，双指针总移动次数也是线性级，因此单次为 $O(n)$。总时间复杂度：$O(n)$。空间复杂度：$O(n)$。

## 3901 - Good Subsequence Queries

**题意**

题目链接：[3901. Good Subsequence Queries](https://leetcode.com/problems/good-subsequence-queries/)

给定数组 `nums` 与整数 `p`。非空子序列若满足长度严格小于 `n` 且元素 gcd 恰好为 `p`，则称为 good。每次查询将 `nums[ind]` 改为 `val`，问修改后是否存在任意 good 子序列，最终统计存在的查询次数。

**解题思路**

**方法一（线段树维护状态）**

先做归一化：只关心能被 `p` 整除的元素。

1. 对区间节点维护 $g0$：把 `v%p!=0` 记为 `0` 后的整体 gcd，用于处理 `cnt<n` 的判定。
2. 维护 $g1$：把 `v%p!=0` 记为 `1` 后的整体 gcd，作为“删一元素状态”合并的基础值。
3. 维护集合 $s$：区间内“删掉恰好一个元素”后，剩余元素 gcd 的所有可能值。

设父区间由左右子区间 $L,R$ 组成，则：

$$
\begin{aligned}
g0 &= \gcd(g0_L, g0_R),\\
g1 &= \gcd(g1_L, g1_R),\\
s &= \{\gcd(x, g1_R)\mid x\in s_L\}\ \cup\ \{\gcd(g1_L, y)\mid y\in s_R\}.
\end{aligned}
$$

查询判定：

1. 若当前可选元素个数 `cnt < n`，存在 good 当且仅当 `root.g0 == 1`。
2. 若 `cnt == n`，需要能删一个元素后 gcd 变为 `1`，即 `1 in root.s`。

**代码（方法一）**

```cpp
class Solution {
public:
    struct Nod {
        int g0 = 0;
        int g1 = 1;
        vector<int> s;
    };

    int n, p, cnt;
    vector<int> a;
    vector<Nod> tr;

    // 构造叶子节点
    Nod mk(int v) {
        Nod x;
        if (v % p == 0) {
            int t = v / p;
            x.g0 = t;
            x.g1 = t;
        } else {
            x.g0 = 0;
            x.g1 = 1;
        }
        x.s = {0};
        return x;
    }

    // 合并两个节点
    Nod mg(const Nod& l, const Nod& r) {
        Nod x;
        x.g0 = gcd(l.g0, r.g0);
        x.g1 = gcd(l.g1, r.g1);

        vector<int> t;
        t.reserve(l.s.size() + r.s.size());

        for (int v : l.s) {
            t.push_back(gcd(v, r.g1));
        }
        for (int v : r.s) {
            t.push_back(gcd(l.g1, v));
        }

        sort(t.begin(), t.end());
        t.erase(unique(t.begin(), t.end()), t.end());
        x.s = move(t);
        return x;
    }

    // 建树
    void bd(int u, int l, int r) {
        if (l == r) {
            tr[u] = mk(a[l]);
            return;
        }
        int m = (l + r) >> 1;
        bd(u << 1, l, m);
        bd(u << 1 | 1, m + 1, r);
        tr[u] = mg(tr[u << 1], tr[u << 1 | 1]);
    }

    // 单点修改
    void up(int u, int l, int r, int pz, int v) {
        if (l == r) {
            tr[u] = mk(v);
            return;
        }
        int m = (l + r) >> 1;
        if (pz <= m) up(u << 1, l, m, pz, v);
        else up(u << 1 | 1, m + 1, r, pz, v);
        tr[u] = mg(tr[u << 1], tr[u << 1 | 1]);
    }

    // 当前数组是否存在 good 子序列
    bool ok() {
        if (cnt < n) {
            return tr[1].g0 == 1;
        }
        for (int v : tr[1].s) {
            if (v == 1) return true;
        }
        return false;
    }

    int countGoodSubseq(vector<int>& nums, int pp, vector<vector<int>>& queries) {
        a = nums;
        p = pp;
        n = a.size();
        tr.assign(4 * n + 5, Nod());

        cnt = 0;
        for (int v : a) {
            if (v % p == 0) cnt++;
        }

        bd(1, 0, n - 1);

        int ans = 0;
        for (auto& q : queries) {
            int id = q[0], v = q[1];

            if (a[id] % p == 0) cnt--;
            if (v % p == 0) cnt++;

            a[id] = v;
            up(1, 0, n - 1, id, v);

            if (ok()) ans++;
        }
        return ans;
    }
};
```

**复杂度分析（方法一）**

建树复杂度约为 $O(n\cdot D)$，单次修改为 $O(D\log n)$，其中 $D$ 为节点状态集合平均规模（本题通常不大，但常数偏大）。空间复杂度：$O(n\cdot D)$。

**方法二（质因子计数 + 异或定位）**

同样先归一化。设当前可选元素个数为 $m$。

1. 当 $m<n$ 时，只要所有可选元素整体 gcd 为 `1` 即可。
2. 当 $m=n$ 时，需要存在某个下标 `i`，删除它后其余元素 gcd 为 `1`。

维护 `cnt[d]`：质因子 `d` 在多少个归一化数中出现，及 `fr[k]`：出现次数为 `k` 的质因子个数。

1. 对 $m<n$，存在 good 等价于 `fr[m] == 0`。
2. 对 $m=n$，先要 `fr[n] == 0`（整体 gcd 为 1）。

再处理“删一个元素后仍 gcd>1”的阻塞点：

1. 若某质因子 `d` 的出现次数为 `n-1`，它只缺一个下标。
2. 维护 `xr[d]` 为包含该质因子的下标异或和，`all` 为全下标异或和，则缺失下标是 `all ^ xr[d]`。
3. 用 `bad[i]` 统计位置 `i` 被多少个 `n-1` 质因子阻塞，`bdc` 统计坏点数量。

于是当 $m=n$ 时，存在 good 当且仅当 `fr[n] == 0 && bdc < n`。

单次修改仅影响旧值与新值的若干不同质因子，常数很小。

**代码（方法二）**

```cpp
class Solution {
public:
    static const int M = 50000;

    int n, p, m, all, bdc;
    vector<int> a, cnt, xr, fr, bad;

    // 返回最小质因子表（只预处理一次）
    vector<int>& gsp() {
        static vector<int> sp = []() {
            vector<int> sp(M + 1);
            for (int i = 0; i <= M; i++) sp[i] = i;
            for (int i = 2; i * i <= M; i++) {
                if (sp[i] == i) {
                    for (int j = i * i; j <= M; j += i) {
                        if (sp[j] == j) sp[j] = i;
                    }
                }
            }
            return sp;
        }();
        return sp;
    }

    // 给 bad[idx] 增减，并维护坏点总数
    void chb(int idx, int dlt) {
        if (bad[idx] == 0 && dlt > 0) bdc++;
        bad[idx] += dlt;
        if (bad[idx] == 0 && dlt < 0) bdc--;
    }

    // 删除一个质因子 d 在位置 idx 的贡献
    void delp(int idx, int d) {
        int old = cnt[d];
        if (old == n - 1) {
            int id = all ^ xr[d];
            chb(id, -1);
        }
        if (old > 0) fr[old]--;

        cnt[d]--;
        xr[d] ^= idx;

        int now = cnt[d];
        if (now > 0) fr[now]++;
        if (now == n - 1) {
            int id = all ^ xr[d];
            chb(id, 1);
        }
    }

    // 添加一个质因子 d 在位置 idx 的贡献
    void addp(int idx, int d) {
        int old = cnt[d];
        if (old == n - 1) {
            int id = all ^ xr[d];
            chb(id, -1);
        }
        if (old > 0) fr[old]--;

        cnt[d]++;
        xr[d] ^= idx;

        int now = cnt[d];
        if (now > 0) fr[now]++;
        if (now == n - 1) {
            int id = all ^ xr[d];
            chb(id, 1);
        }
    }

    // 枚举 x 的不同质因子，并执行增删操作
    void upd(int idx, int x, int tp) {
        auto& sp = gsp();
        while (x > 1) {
            int d = sp[x];
            if (tp == 1) addp(idx, d);
            else delp(idx, d);
            while (x % d == 0) x /= d;
        }
    }

    // 当前数组是否存在 good 子序列
    bool ok() {
        if (m == 0) return false;
        if (m < n) {
            return fr[m] == 0;
        }
        return fr[n] == 0 && bdc < n;
    }

    int countGoodSubseq(vector<int>& nums, int pp, vector<vector<int>>& queries) {
        a = nums;
        p = pp;
        n = a.size();

        cnt.assign(M + 1, 0);
        xr.assign(M + 1, 0);
        fr.assign(n + 1, 0);
        bad.assign(n, 0);

        m = 0;
        bdc = 0;
        all = 0;
        for (int i = 0; i < n; i++) all ^= i;

        for (int i = 0; i < n; i++) {
            if (a[i] % p == 0) {
                m++;
                upd(i, a[i] / p, 1);
            }
        }

        int ans = 0;
        for (auto& q : queries) {
            int idx = q[0];
            int val = q[1];

            if (a[idx] % p == 0) {
                m--;
                upd(idx, a[idx] / p, -1);
            }

            a[idx] = val;

            if (a[idx] % p == 0) {
                m++;
                upd(idx, a[idx] / p, 1);
            }

            if (ok()) ans++;
        }

        return ans;
    }
};
```

**复杂度分析（方法二）**

设一个数的不同质因子个数为 $\omega(x)$。

预处理 SPF 为 $O(V\log\log V)$（$V=5\times10^4$，仅一次）。

时间复杂度：$O(V\log\log V + (n+q)\cdot\omega(V))$，其中单次查询为 $O(\omega(\text{old}/p)+\omega(\text{new}/p))$。

空间复杂度：$O(V+n)$。
