## 3912 - Valid Elements in an Array

**题意**

题目链接：[3912. Valid Elements in an Array](https://leetcode.com/problems/valid-elements-in-an-array/)

给定整数数组 `nums`，长度为 $n$。元素 `nums[i]` 合法当且仅当它严格大于左边所有元素，或严格大于右边所有元素。首尾元素一定合法。返回所有合法元素，保持原相对顺序。

**解题思路**

对每个位置 $i$ 需要知道：左边所有元素的最大值、右边所有元素的最大值。

预处理 `pre[i]` 为 $nums[0..i-1]$ 的最大值，`suf[i]` 为 $nums[i+1..n-1]$ 的最大值。遍历时若 $nums[i] > pre[i]$ 或 $nums[i] > suf[i]$ 则加入答案。由于 $nums[i] \ge 1$ 而 $pre[0]=suf[n-1]=0$，首尾元素自然满足。

**代码**

```cpp
class Solution {
public:
    vector<int> findValidElements(vector<int>& nums) {
        int n = nums.size();
        vector<int> pre(n, 0), suf(n, 0);
        for (int i = 1; i < n; i++) pre[i] = max(pre[i - 1], nums[i - 1]);
        for (int i = n - 2; i >= 0; i--) suf[i] = max(suf[i + 1], nums[i + 1]);
        vector<int> ans;
        for (int i = 0; i < n; i++) {
            if (nums[i] > pre[i] || nums[i] > suf[i])
                ans.push_back(nums[i]);
        }
        return ans;
    }
};
```

**复杂度分析**

时间复杂度：$O(n)$，空间复杂度：$O(n)$。

## 3913 - Sort Vowels by Frequency

**题意**

题目链接：[3913. Sort Vowels by Frequency](https://leetcode.com/problems/sort-vowels-by-frequency/)

给定小写字符串 `s`，只允许重新排列其中的元音字母（`a, e, i, o, u`），使得元音按出现频率降序排列；频率相同时按首次出现位置升序排列。返回修改后的字符串。

**解题思路**

元音只有 5 种，统计每个元音的出现次数 `cnt` 和首次出现位置 `fst`。将出现过的元音取出，按 `(cnt 大者在前，cnt 相同时 fst 小者在前)` 排序，再按对应次数展开成序列。最后从左到右扫描原串，在元音位置依次填入新序列即可。

**代码**

```cpp
class Solution {
public:
    bool vow(char c) {
        return c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u';
    }
    string sortVowels(string s) {
        int n = s.size();
        vector<int> cnt(26, 0), fst(26, n);
        for (int i = 0; i < n; i++) {
            if (vow(s[i])) {
                int x = s[i] - 'a';
                cnt[x]++;
                fst[x] = min(fst[x], i);
            }
        }
        string vs = "aeiou";
        vector<char> ord;
        for (char c : vs) {
            if (cnt[c - 'a'] > 0) ord.push_back(c);
        }
        sort(ord.begin(), ord.end(), [&](char x, char y) {
            int a = x - 'a', b = y - 'a';
            if (cnt[a] != cnt[b]) return cnt[a] > cnt[b];
            return fst[a] < fst[b];
        });
        string t;
        for (char c : ord) {
            int x = c - 'a';
            t.append(cnt[x], c);
        }
        int p = 0;
        for (int i = 0; i < n; i++) {
            if (vow(s[i])) s[i] = t[p++];
        }
        return s;
    }
};
```

**复杂度分析**

时间复杂度：$O(n)$，空间复杂度：$O(n)$。

## 3914 - Minimum Operations to Make Array Non Decreasing

**题意**

题目链接：[3914. Minimum Operations to Make Array Non Decreasing](https://leetcode.com/problems/minimum-operations-to-make-array-non-decreasing/)

给定数组 `nums`，一次操作可选择任意子数组并给其中每个元素加上任意正整数 $x$（代价为 $x$，不是 $x \times$ 长度）。要求通过若干次操作使数组变为非递减数组，最小化所有 $x$ 的总和。

**解题思路**

关键观察：若相邻 $nums[i] > nums[i+1]$，设下降量为 $d = nums[i] - nums[i+1]$，则最终数组中 $i+1$ 相对于 $i$ 必须多增加至少 $d$，因此每个下降位置至少贡献 $d$ 的必要代价。

下界可达到：对每个下降位置 $i$，对后缀 $[i+1, n-1]$ 增加恰好 $d$。后缀操作不破坏已修复的非递减关系。总代价正好等于所有相邻下降量之和。

**代码**

```cpp
class Solution {
public:
    long long minOperations(vector<int>& nums) {
        int n = nums.size();
        long long ans = 0;
        for (int i = 0; i + 1 < n; i++) {
            if (nums[i] > nums[i + 1])
                ans += nums[i] - nums[i + 1];
        }
        return ans;
    }
};
```

**复杂度分析**

时间复杂度：$O(n)$，空间复杂度：$O(1)$。

## 3915 - Maximum Sum of Alternating Subsequence With Distance at Least K

**题意**

题目链接：[3915. Maximum Sum of Alternating Subsequence With Distance at Least K](https://leetcode.com/problems/maximum-sum-of-alternating-subsequence-with-distance-at-least-k/)

给定数组 `nums` 和整数 $k$，选出下标严格递增的子序列，满足相邻下标距离至少为 $k$，且选中数值严格交替（峰-谷-峰或谷-峰-谷）。子序列长度 $1$ 亦合法。求最大元素和。

---

### 解法一：直接值域树状数组

**思路**

定义两种 DP 状态：
- $up_i$：以 $i$ 结尾、最后一步为上升的最大和
- $dn_i$：以 $i$ 结尾、最后一步为下降的最大和

转移时前驱 $j$ 必须满足 $j \le i-k$，且交替要求：
$$up_i = nums_i + \max\{dn_j \mid nums_j < nums_i, j \le i-k\}$$
$$dn_i = nums_i + \max\{up_j \mid nums_j > nums_i, j \le i-k\}$$

扫描时将位置 $i-k$ 的状态加入数据结构。需要快速查询「值小于 $x$ 的最大 $dn$」和「值大于 $x$ 的最大 $up$」，分别用两个树状数组维护。

其中「值小于 $x$」直接用 `bl.qry(x-1)` 查询前缀最大值；而「值大于 $x$」通过**值域翻转**转化为前缀查询：设翻转映射 $rev = M - v + 1$，则原空间中的 $v_1 > v_2$ 在翻转后变成 $rev_1 < rev_2$，于是查询「$v > x$」等价于在翻转后的空间中查询「$rev < M - x + 1$」。值域固定为 $10^5$，直接建立长度为 $M=100000$ 的树状数组。

**代码**

```cpp
class Solution {
public:
    struct BIT {
        int n;
        vector<long long> tr;
        BIT(int n = 0) { init(n); }
        void init(int m) { n = m; tr.assign(n + 1, 0); }
        void upd(int x, long long v) {
            for (int i = x; i <= n; i += i & -i) tr[i] = max(tr[i], v);
        }
        long long qry(int x) {
            long long res = 0;
            for (int i = x; i > 0; i -= i & -i) res = max(res, tr[i]);
            return res;
        }
    };

    long long maxAlternatingSum(vector<int>& nums, int k) {
        int n = nums.size();
        int M = 100000;
        vector<long long> up(n, 0), dn(n, 0);
        BIT bl(M), bg(M);
        long long ans = 0;
        for (int i = 0; i < n; i++) {
            int j = i - k;
            if (j >= 0) {
                int x = nums[j];
                bl.upd(x, dn[j]);
                bg.upd(M - x + 1, up[j]);
            }
            int x = nums[i];
            up[i] = bl.qry(x - 1) + x;
            dn[i] = bg.qry(M - x) + x;
            ans = max(ans, max(up[i], dn[i]));
        }
        return ans;
    }
};
```

**复杂度**

时间复杂度：$O(n \log M) = O(n \log 10^5)$，空间复杂度：$O(n + M)$。

### 解法二：坐标压缩优化

值域固定 $10^5$ 偏大，且实际参与的值最多只有 $n$ 个。先对 `nums` 去重排序，将值域压缩到 $[1, m]$（$m \le n$）。树状数组规模从 $10^5$ 降至 $m$，常数显著降低。

查询逻辑与解法一一致：$up_i$ 用前缀最大值查询，$dn_i$ 通过值域翻转 $rev = m - x + 1$ 转化为前缀查询。只需在 DP 前增加坐标压缩：

```cpp
vector<int> val = nums;
sort(val.begin(), val.end());
val.erase(unique(val.begin(), val.end()), val.end());
int m = val.size();
vector<int> rk(n);
for (int i = 0; i < n; i++)
    rk[i] = lower_bound(val.begin(), val.end(), nums[i]) - val.begin() + 1;
```

之后所有树状数组操作均基于压缩后的坐标 $rk[i]$。

**完整代码**

```cpp
class Solution {
public:
    struct BIT {
        int n;
        vector<long long> tr;
        BIT(int n = 0) { init(n); }
        void init(int m) { n = m; tr.assign(n + 1, 0); }
        void upd(int x, long long v) {
            for (int i = x; i <= n; i += i & -i) tr[i] = max(tr[i], v);
        }
        long long qry(int x) {
            long long res = 0;
            for (int i = x; i > 0; i -= i & -i) res = max(res, tr[i]);
            return res;
        }
    };

    long long maxAlternatingSum(vector<int>& nums, int k) {
        int n = nums.size();
        vector<int> val = nums;
        sort(val.begin(), val.end());
        val.erase(unique(val.begin(), val.end()), val.end());
        int m = val.size();
        vector<int> rk(n);
        for (int i = 0; i < n; i++)
            rk[i] = lower_bound(val.begin(), val.end(), nums[i]) - val.begin() + 1;

        vector<long long> up(n), dn(n);
        BIT bl(m), bg(m);
        long long ans = 0;

        for (int i = 0; i < n; i++) {
            int j = i - k;
            if (j >= 0) {
                int x = rk[j];
                bl.upd(x, dn[j]);
                bg.upd(m - x + 1, up[j]);
            }
            int x = rk[i];
            up[i] = bl.qry(x - 1) + nums[i];
            int rx = m - x + 1;
            dn[i] = bg.qry(rx - 1) + nums[i];
            ans = max(ans, max(up[i], dn[i]));
        }
        return ans;
    }
};
```

**复杂度**

时间复杂度：$O(n \log n)$（含排序与坐标压缩），空间复杂度：$O(n + m)$。当重复值较多时 $m \ll M$，实际运行更快。
