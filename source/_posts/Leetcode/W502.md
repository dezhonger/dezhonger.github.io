## A - Check Adjacent Digit Differences

**题意**

题目链接：[Check Adjacent Digit Differences](https://leetcode.com/problems/check-adjacent-digit-differences/)

给定一个只包含数字字符的字符串 `s`。

判断：对于所有相邻位置 $i-1$ 和 $i$，是否都有 $|s[i] - s[i-1]| \le 2$。若全部满足返回 `true`，否则返回 `false`。

- $2 \le s.length \le 100$
- `s` 只包含数字字符

**解题思路**

从左到右遍历字符串，检查每一对相邻字符。由于字符 `'0'` 到 `'9'` 在 ASCII 中连续，直接相减即可得到数值差。若某对相邻数字的绝对值差大于 $2$，立即返回 `false`。遍历结束说明所有相邻数字都满足条件，返回 `true`。

**代码**

```cpp
class Solution {
public:
    bool checkAdjacentDigitDifferenceAtMostTwo(string s) {
        int n = s.size();
        for (int i = 1; i < n; i++) {
            if (abs(s[i] - s[i - 1]) > 2) {
                return false;
            }
        }
        return true;
    }
};
```

**复杂度分析**

时间复杂度：$O(n)$，只需遍历一遍字符串。

空间复杂度：$O(1)$，只使用了常数个额外变量。

**变形**

如果把条件改为「相邻数字差不超过 $k$」，只需把判断条件改成 `d > k`。当数据范围扩大到 $n \le 10^7$ 时，该做法仍然最优，因为至少需要检查所有相邻位置，时间复杂度下界就是 $O(n)$。

## B - Count K-th Roots in a Range

**题意**

题目链接：[Count K-th Roots in a Range](https://leetcode.com/problems/count-k-th-roots-in-a-range/)

给定三个整数 `l`、`r`、`k`。统计区间 $[l, r]$ 中有多少个整数 $y$ 满足存在整数 $x$ 使得 $y = x^k$。

- $0 \le l \le r \le 10^9$
- $1 \le k \le 30$

**解题思路**

先求 $[0, n]$ 中有多少个 perfect k-th power。若 $x \ge 0$，则 $x^k \le n$ 满足条件的 $x$ 数量为 $\lfloor \sqrt[k]{n} \rfloor + 1$，其中 $+1$ 是因为 $x = 0$ 也产生 $0^k = 0$。

答案转化为 `count(r) - count(l - 1)`，其中 `count(n) = floor_kth_root(n, k) + 1`。

求 $\lfloor \sqrt[k]{n} \rfloor$ 使用二分：在 $[0, n]$ 范围内二分最大的 $x$，满足 $x^k \le n$。在计算 $x^k$ 时，用乘法前除法判断溢出避免超范围。

**代码**

```cpp
class Solution {
public:
    bool checkPowerLe(long long a, int k, long long n) {
        if (a == 0) return 0 <= n;
        long long res = 1;
        for (int i = 0; i < k; i++) {
            if (res > n / a) return false;
            res *= a;
        }
        return res <= n;
    }

    long long floorKthRoot(long long n, int k) {
        long long l = 0, r = n, ans = 0;
        while (l <= r) {
            long long mid = (l + r) / 2;
            if (checkPowerLe(mid, k, n)) {
                ans = mid;
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return ans;
    }

    long long countKthPowers(long long n, int k) {
        if (n < 0) return 0;
        return floorKthRoot(n, k) + 1;
    }

    int countKthRootsInRange(int l, int r, int k) {
        long long ans = countKthPowers(r, k) - countKthPowers((long long)l - 1, k);
        return (int)ans;
    }
};
```

**复杂度分析**

设值域上界为 $V = r$。二分需要 $O(\log V)$ 次，每次判断 $mid^k \le n$ 最多乘 $k$ 次。

时间复杂度：$O(k \log V)$。

空间复杂度：$O(1)$。

**变形**

如果 $r \le 10^{18}$，二分做法仍然适用，只需继续使用乘法前除法判断溢出。如果需要多次询问同一个 $k$，可以预处理所有不超过上限的 $x^k$，排序去重后用二分统计区间内数量。

## C - Largest Local Values in a Matrix II

**题意**

题目链接：[Largest Local Values in a Matrix II](https://leetcode.com/problems/largest-local-values-in-a-matrix-ii/)

给定一个 $n \times m$ 的非负整数矩阵 `matrix`。

对于一个非零单元格 $(row, col)$，设 $x = matrix[row][col]$。它检查所有满足 $|r - row| \le x$ 且 $|c - col| \le x$ 的单元格，但忽略矩阵范围外的单元格以及同时满足 $|r - row| = x$ 且 $|c - col| = x$ 的单元格（即正方形的四个角）。

如果所有被考虑的单元格中，没有单元格的值大于 $x$，则 $(row, col)$ 是 local maximum。返回 matrix 中 local maximum 的数量。

- $1 \le n, m \le 200$
- $0 \le matrix[i][j] \le 200$

**解题思路**

矩阵元素值域为 $[0, 200]$，很小。可以按值 $x$ 枚举，建立二维前缀和 `pre`，其中 `pre[i][j]` 表示前缀区域内有多少个单元格的值 $> x$。

对于值为 $x$ 的格子 $(i, j)$，它是否是 local maximum，只需判断它的检查范围内是否存在值 $> x$ 的格子。这个数量可以通过二维前缀和 $O(1)$ 查询。

但题目要求忽略四个角：$(i-x, j-x)$、$(i-x, j+x)$、$(i+x, j-x)$、$(i+x, j+x)$。如果这些角在矩阵内并且值大于 $x$，需要从计数中减掉。减掉四个角之后，若剩余的「大于 $x$ 的格子数量」为 $0$，则该格子是 local maximum。

**代码**

```cpp
class Solution {
public:
    int queryPrefixSum(const vector<vector<int>>& pre, int r1, int c1, int r2, int c2) {
        return pre[r2 + 1][c2 + 1] - pre[r1][c2 + 1] - pre[r2 + 1][c1] + pre[r1][c1];
    }

    int countLocalMaximumValuesInMatrix(vector<vector<int>>& matrix) {
        const int MAX_VAL = 200;
        int n = matrix.size(), m = matrix[0].size();
        int ans = 0;

        for (int x = 1; x <= MAX_VAL; x++) {
            vector<vector<int>> pre(n + 1, vector<int>(m + 1, 0));
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < m; j++) {
                    int v = matrix[i][j] > x;
                    pre[i + 1][j + 1] = pre[i][j + 1] + pre[i + 1][j] - pre[i][j] + v;
                }
            }

            for (int i = 0; i < n; i++) {
                for (int j = 0; j < m; j++) {
                    if (matrix[i][j] != x) continue;

                    int r1 = max(0, i - x), r2 = min(n - 1, i + x);
                    int c1 = max(0, j - x), c2 = min(m - 1, j + x);
                    int cnt = queryPrefixSum(pre, r1, c1, r2, c2);

                    for (int dx : {-x, x}) {
                        for (int dy : {-x, x}) {
                            int r = i + dx, c = j + dy;
                            if (r >= 0 && r < n && c >= 0 && c < m && matrix[r][c] > x) {
                                cnt--;
                            }
                        }
                    }

                    if (cnt == 0) ans++;
                }
            }
        }
        return ans;
    }
};
```

**复杂度分析**

设 $n = matrix.length$，$m = matrix[0].length$，$V = 200$。每个值 $x$ 都会构造一次二维前缀和并扫描一次矩阵。

时间复杂度：$O(V \cdot n \cdot m)$。由于 $V = 200$，$n, m \le 200$，最多约 $8 \times 10^6$ 次操作，可通过。

空间复杂度：$O(n \cdot m)$，只需维护当前值 $x$ 对应的二维前缀和。

**变形**

如果矩阵元素范围扩大到 $10^9$，不能再枚举所有值。此时可以考虑使用二维 Sparse Table 支持查询矩形最大值，查询整个正方形区域的最大值，再额外处理被忽略的四个角。

## D - Smallest Unique Subarray

**题意**

题目链接：[Smallest Unique Subarray](https://leetcode.com/problems/smallest-unique-subarray/)

给定一个整数数组 `nums`。找到最短的非空连续子数组，使得它在所有同样长度的子数组中只出现一次。返回最短唯一子数组的长度。

- $1 \le nums.length \le 10^5$
- $1 \le nums[i] \le 10^5$

**解题思路**

任意子数组 `nums[i ... j]` 都可以看成从位置 $i$ 开始的后缀的一个前缀。因此问题转化为：对于每个位置 $i$，求从 $i$ 开始的最短唯一前缀长度，最后取所有位置的最小值。

若从位置 $i$ 开始的长度为 $len$ 的子数组不唯一，说明存在另一个位置 $j$，使得 `nums[i ... i+len-1] == nums[j ... j+len-1]`，即后缀 $i$ 和后缀 $j$ 的最长公共前缀长度至少为 $len$。因此，对于后缀 $i$，若它与其他后缀的最大公共前缀长度为 $mx$，则长度 $\le mx$ 的前缀都不唯一，长度 $mx+1$ 的前缀才可能唯一。

后缀数组会把所有后缀按字典序排序。某个后缀和其他后缀的最大 LCP，只需看它在后缀数组中的前一个和后一个后缀。即：
$$mx = \max(LCP(\text{后缀 } i, \text{排名} - 1), LCP(\text{后缀 } i, \text{排名} + 1))$$

答案就是 $\min(mx + 1)$。

下面提供三种解法：后缀数组 + LCP（$O(n \log n)$）、后缀自动机 SAM（期望 $O(n)$）、SA-IS 线性后缀数组（$O(n)$）。

**解法一：后缀数组 + LCP（倍增法）**

```cpp
class Solution {
public:
    vector<int> buildSuffixArray(const vector<int>& a) {
        int n = a.size();
        vector<int> p(n), c(n);
        iota(p.begin(), p.end(), 0);
        sort(p.begin(), p.end(), [&](int x, int y) { return a[x] < a[y]; });
        c[p[0]] = 0;
        for (int i = 1; i < n; i++) c[p[i]] = c[p[i-1]] + (a[p[i]] != a[p[i-1]]);

        for (int len = 1; len < n; len <<= 1) {
            vector<int> np(n), nc(n), cnt, pos;
            int lim = c[p[n-1]] + 2;

            cnt.assign(lim, 0);
            for (int id : p) cnt[id + len < n ? c[id + len] + 1 : 0]++;
            pos.assign(lim, 0);
            for (int i = 1; i < lim; i++) pos[i] += pos[i-1] + cnt[i-1];
            for (int id : p) np[pos[id + len < n ? c[id + len] + 1 : 0]++] = id;

            cnt.assign(lim, 0);
            for (int id : np) cnt[c[id] + 1]++;
            pos.assign(lim, 0);
            for (int i = 1; i < lim; i++) pos[i] += pos[i-1] + cnt[i-1];
            for (int id : np) p[pos[c[id] + 1]++] = id;

            nc[p[0]] = 0;
            for (int i = 1, ncl = 1; i < n; i++) {
                int x = p[i-1], y = p[i];
                int x1 = c[x], y1 = c[y];
                int x2 = x + len < n ? c[x + len] : -1;
                int y2 = y + len < n ? c[y + len] : -1;
                if (x1 != y1 || x2 != y2) ncl++;
                nc[y] = ncl - 1;
            }
            c.swap(nc);
            if (c[p[n-1]] == n) break;
        }
        return p;
    }

    vector<int> buildLCPArray(const vector<int>& a, const vector<int>& p) {
        int n = a.size();
        vector<int> rk(n);
        for (int i = 0; i < n; i++) rk[p[i]] = i;
        vector<int> h(max(0, n-1), 0);
        int k = 0;
        for (int i = 0; i < n; i++) {
            int r = rk[i];
            if (r == n-1) { k = 0; continue; }
            int j = p[r + 1];
            while (i + k < n && j + k < n && a[i+k] == a[j+k]) k++;
            h[r] = k;
            if (k) k--;
        }
        return h;
    }

    int findSmallestUniqueSubarray(vector<int>& nums) {
        int n = nums.size();
        vector<int> sa = buildSuffixArray(nums);
        vector<int> lcp = buildLCPArray(nums, sa);
        vector<int> rk(n);
        for (int i = 0; i < n; i++) rk[sa[i]] = i;

        int ans = n;
        for (int i = 0; i < n; i++) {
            int r = rk[i], mx = 0;
            if (r > 0) mx = max(mx, lcp[r-1]);
            if (r + 1 < n) mx = max(mx, lcp[r]);
            int len = mx + 1;
            if (len <= n - i) ans = min(ans, len);
        }
        return ans;
    }
};
```

时间复杂度 $O(n \log n)$，空间复杂度 $O(n)$。

**解法二：后缀自动机 SAM（期望 $O(n)$）**

SAM 中每个状态表示一类子数组，出现次数相同。设状态 $v$ 能表示的子数组长度范围是 $[len[link[v]]+1, len[v]]$。若 $oc[v] = 1$，说明该状态代表的所有子数组只出现一次，因此最短唯一子数组长度为 $len[link[v]] + 1$。答案为所有 $oc[v] = 1$ 状态的 $len[link[v]] + 1$ 最小值。

```cpp
class Solution {
public:
    struct State {
        int len = 0, link = -1, occ = 0;
        unordered_map<int, int> next;
    };

    vector<State> st;
    int last;

    void extend(int c) {
        int cur = st.size();
        st.push_back({});
        st[cur].len = st[last].len + 1;
        st[cur].occ = 1;

        int p = last;
        while (p != -1 && !st[p].next.count(c)) {
            st[p].next[c] = cur;
            p = st[p].link;
        }

        if (p == -1) {
            st[cur].link = 0;
        } else {
            int q = st[p].next[c];
            if (st[p].len + 1 == st[q].len) {
                st[cur].link = q;
            } else {
                int clone = st.size();
                st.push_back(st[q]);
                st[clone].len = st[p].len + 1;
                st[clone].occ = 0;

                while (p != -1 && st[p].next.count(c) && st[p].next[c] == q) {
                    st[p].next[c] = clone;
                    p = st[p].link;
                }
                st[q].link = st[cur].link = clone;
            }
        }
        last = cur;
    }

    int findSmallestUniqueSubarray(vector<int>& nums) {
        st.clear();
        st.push_back({});
        last = 0;

        for (int x : nums) extend(x);

        int sz = st.size();
        int mx = 0;
        for (auto& s : st) mx = max(mx, s.len);

        vector<int> cnt(mx + 1, 0);
        for (auto& s : st) cnt[s.len]++;
        for (int i = 1; i <= mx; i++) cnt[i] += cnt[i-1];

        vector<int> order(sz);
        for (int i = sz - 1; i >= 0; i--) order[--cnt[st[i].len]] = i;

        for (int i = sz - 1; i > 0; i--) {
            int v = order[i];
            int p = st[v].link;
            if (p != -1) st[p].occ += st[v].occ;
        }

        int ans = nums.size();
        for (int v = 1; v < sz; v++) {
            if (st[v].occ == 1) {
                int p = st[v].link;
                ans = min(ans, st[p].len + 1);
            }
        }
        return ans;
    }
};
```

时间复杂度期望 $O(n)$，空间复杂度 $O(n)$。

**解法三：SA-IS 线性后缀数组 + Kasai（严格 $O(n)$）**

SA-IS 是线性时间后缀数组构造算法，对整数字母表复杂度为 $O(n + A)$（$A$ 为值域大小）。配合 Kasai 算法求 LCP，整体严格 $O(n)$。

```cpp
class Solution {
public:
    vector<int> sais(const vector<int>& s, int lim) {
        int n = s.size();
        if (n <= 2) {
            if (n == 1) return {0};
            return s[0] < s[1] ? vector<int>{0,1} : vector<int>{1,0};
        }

        vector<int> sa(n, -1), typ(n, 0);
        for (int i = n - 2; i >= 0; i--)
            typ[i] = (s[i] == s[i+1]) ? typ[i+1] : (s[i] < s[i+1]);

        vector<int> sl(lim + 2, 0), ss(lim + 2, 0);
        for (int i = 0; i < n; i++) {
            if (!typ[i]) ss[s[i]]++;
            else sl[s[i] + 1]++;
        }
        for (int i = 0; i <= lim; i++) {
            ss[i] += sl[i];
            sl[i + 1] += ss[i];
        }

        auto inducedSort = [&](const vector<int>& lms) {
            fill(sa.begin(), sa.end(), -1);
            vector<int> buf(lim + 2);
            copy(ss.begin(), ss.end(), buf.begin());
            for (int x : lms) {
                if (x != n) sa[buf[s[x]]++] = x;
            }
            copy(sl.begin(), sl.end(), buf.begin());
            sa[buf[s[n-1]]++] = n - 1;
            for (int i = 0; i < n; i++) {
                int v = sa[i];
                if (v >= 1 && !typ[v-1]) sa[buf[s[v-1]]++] = v - 1;
            }
            copy(sl.begin(), sl.end(), buf.begin());
            for (int i = n - 1; i >= 0; i--) {
                int v = sa[i];
                if (v >= 1 && typ[v-1]) sa[--buf[s[v-1] + 1]] = v - 1;
            }
        };

        vector<int> mp(n + 1, -1);
        int m = 0;
        for (int i = 1; i < n; i++)
            if (!typ[i-1] && typ[i]) mp[i] = m++;

        vector<int> lms;
        for (int i = 1; i < n; i++)
            if (!typ[i-1] && typ[i]) lms.push_back(i);

        inducedSort(lms);

        if (m > 0) {
            vector<int> slm;
            for (int x : sa) if (mp[x] != -1) slm.push_back(x);

            vector<int> rs(m, 0);
            rs[mp[slm[0]]] = 0;
            for (int i = 1; i < m; i++) {
                int l = slm[i-1], r = slm[i];
                int el = mp[l] + 1 < m ? lms[mp[l] + 1] : n;
                int er = mp[r] + 1 < m ? lms[mp[r] + 1] : n;
                bool same = true;
                if (el - l != er - r) same = false;
                else {
                    while (l < el) {
                        if (s[l] != s[r]) break;
                        l++; r++;
                    }
                    if (l == n || s[l] != s[r]) same = false;
                }
                if (!same) rs[mp[slm[i]]] = rs[mp[slm[i-1]]] + 1;
                else rs[mp[slm[i]]] = rs[mp[slm[i-1]]];
            }

            vector<int> rsa = sais(rs, rs[mp[slm[m-1]]]);
            for (int i = 0; i < m; i++) slm[i] = lms[rsa[i]];
            inducedSort(slm);
        }

        return sa;
    }

    vector<int> buildSuffixArray(const vector<int>& nums) {
        int n = nums.size();
        int mx = 0;
        for (int x : nums) mx = max(mx, x);
        vector<int> s(n + 1);
        for (int i = 0; i < n; i++) s[i] = nums[i];
        s[n] = 0;

        vector<int> sa = sais(s, mx);
        vector<int> res;
        for (int x : sa) if (x < n) res.push_back(x);
        return res;
    }

    vector<int> buildLCPArray(const vector<int>& nums, const vector<int>& sa) {
        int n = nums.size();
        vector<int> rk(n);
        for (int i = 0; i < n; i++) rk[sa[i]] = i;
        vector<int> h(max(0, n-1), 0);
        int k = 0;
        for (int i = 0; i < n; i++) {
            int r = rk[i];
            if (r == n-1) { k = 0; continue; }
            int j = sa[r + 1];
            while (i + k < n && j + k < n && nums[i+k] == nums[j+k]) k++;
            h[r] = k;
            if (k) k--;
        }
        return h;
    }

    int findSmallestUniqueSubarray(vector<int>& nums) {
        int n = nums.size();
        vector<int> sa = buildSuffixArray(nums);
        vector<int> lcp = buildLCPArray(nums, sa);
        vector<int> rk(n);
        for (int i = 0; i < n; i++) rk[sa[i]] = i;

        int ans = n;
        for (int i = 0; i < n; i++) {
            int r = rk[i], mx = 0;
            if (r > 0) mx = max(mx, lcp[r-1]);
            if (r + 1 < n) mx = max(mx, lcp[r]);
            int len = mx + 1;
            if (len <= n - i) ans = min(ans, len);
        }
        return ans;
    }
};
```

时间复杂度 $O(n + A)$，空间复杂度 $O(n + A)$。本题 $A \le 10^5$，与 $n$ 同级，可视为线性。

**复杂度分析**

三种解法的复杂度总结如下：

| 解法 | 时间复杂度 | 空间复杂度 |
| --- | --- | --- |
| 后缀数组 + LCP | $O(n \log n)$ | $O(n)$ |
| 后缀自动机 SAM | 期望 $O(n)$ | $O(n)$ |
| SA-IS + Kasai | $O(n + A)$ | $O(n + A)$ |

**变形**

如果需要求每个位置开始的最短唯一子数组长度，只需把每个位置的 $len = mx + 1$ 存下来即可。如果需要统计长度为 $len$ 的唯一子数组数量，可以用哈希或者后缀数组统计相邻 LCP，判断哪些长度为 $len$ 的子数组只出现一次。
