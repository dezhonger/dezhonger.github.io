## 题意

给定长度为 $n$ 的整数序列 $x_1, x_2, \dots, x_n$，满足：

- $x_1 = 2$
- 对所有 $1 < i \le n$，有 $x_{i-1} < x_i$
- 对所有 $1 \le i, j \le n$，有
  $$
  x_i^j < (x_j + 1)^i
  $$

记满足条件的序列个数为 $t(n)$。已知：

- $t(10)=86195$
- $t(20)=5227991891$

求
$$
t(10^{10}) \bmod 10^9.
$$

## 解题思路

### 一、等价转化

这是本题第一处关键转化。

将
$$
x_i^j < (x_j + 1)^i
$$
两边同时取 $ij$ 次方根（两边均为正数），得到
$$
x_i^{1/i} < (x_j + 1)^{1/j}.
$$

记
$$
L_i = x_i^{1/i}, \qquad R_i = (x_i + 1)^{1/i},
$$
则条件变成
$$
L_i < R_j \qquad (\forall i,j).
$$

现在考虑开区间 $(L_i, R_i)$。一组开区间有公共交集，当且仅当
$$
\max_i L_i < \min_i R_i.
$$
而由 $L_i < R_j\ (\forall i,j)$ 可直接推出上式：取 $A=\max_i L_i$，它是某个 $L_{i_0}$；因为对任意 $j$ 都有 $L_{i_0}<R_j$，故
$$
A < \min_j R_j.
$$

因此存在某个实数
$$
\alpha \in \bigcap_{i=1}^n (L_i, R_i),
$$
即对每个 $i$ 都有
$$
x_i^{1/i} < \alpha < (x_i + 1)^{1/i}
\iff
x_i < \alpha^i < x_i + 1.
$$
所以
$$
x_i = \lfloor \alpha^i \rfloor.
$$

反向也成立：若存在 $\alpha$ 使 $x_i=\lfloor \alpha^i \rfloor$，则
$$
x_i < \alpha^i, \qquad \alpha^j < x_j + 1
\Longrightarrow
x_i^j < \alpha^{ij} < (x_j + 1)^i.
$$
所以原条件与“存在同一个 $\alpha$ 使 $x_i = \lfloor \alpha^i \rfloor$”完全等价。

又因 $x_1=2$，立刻得到 $2<\alpha<3$。

### 二、断点计数

定义
$$
F_n(\alpha)=\bigl(\lfloor \alpha \rfloor,\lfloor \alpha^2 \rfloor,\dots,\lfloor \alpha^n \rfloor\bigr),\qquad \alpha\in(2,3).
$$
则 $t(n)$ 就是 $F_n$ 在 $(2,3)$ 上能取到的不同值个数。

关键观察：

- $\lfloor \alpha^k \rfloor$ 只有在 $\alpha^k$ 穿过整数时才会变化。
- 因此整个序列只会在满足
  $$
  \exists\,k\le n,\ \alpha^k\in\mathbb{Z}
  $$
  的点发生变化。

把这类 $\alpha$ 称为断点。于是：

- 在任意两个相邻断点之间的开区间内，所有 $\lfloor \alpha^k \rfloor$ 都不变，$F_n$ 常值。
- 跨过任一断点，至少有某个 $\lfloor \alpha^k \rfloor$ 跳变，$F_n$ 必变。

所以“不同序列个数 = 断点个数 + 1”：
$$
t(n)=1+\#\{\alpha\in(2,3)\mid \exists\,k\le n,\ \alpha^k\in\mathbb{Z}\}.
$$

### 三、最小幂次

定义
$$
\operatorname{ord}(\alpha)=\min\{r\ge1:\alpha^r\in\mathbb{Z}\},
$$
并令
$$
f(k)=\#\{\alpha\in(2,3):\operatorname{ord}(\alpha)=k\}.
$$
则
$$
t(n)=1+\sum_{k=1}^n f(k).
$$

固定 $k$。若 $\operatorname{ord}(\alpha)=k$，设 $m=\alpha^k\in\mathbb{Z}$，则
$$
2^k < m < 3^k, \qquad \alpha = m^{1/k}.
$$
于是可转化为统计区间 $(2^k,3^k)$ 内哪些整数 $m$ 使得 $\operatorname{ord}(m^{1/k})=k$。

设
$$
m=\prod p_i^{e_i},\qquad g=\gcd(e_1,e_2,\dots).
$$
可证明
$$
\operatorname{ord}(m^{1/k}) = \frac{k}{\gcd(k,g)}.
$$
因此
$$
\operatorname{ord}(m^{1/k})=k
\iff
\gcd(k,g)=1.
$$

再将其翻译为“完全幂”条件：对任意素数 $p$，有
$$
p\mid g \iff m\text{ 是完全 }p\text{ 次幂}.
$$
所以
$$
\gcd(k,g)=1
\iff
\forall p\mid k,\ m\text{ 不是完全 }p\text{ 次幂}.
$$

据此对 $k$ 的质因子做容斥，得到
$$
f(k)=\sum_{d\mid \operatorname{rad}(k)} \mu(d)\bigl(3^{k/d}-2^{k/d}-1\bigr),
$$
其中 $\operatorname{rad}(k)$ 为不同质因子乘积，$\mu$ 为 Möbius 函数。

### 四、快速求和

代回
$$
t(n)=1+\sum_{k=1}^n f(k),
$$
交换求和可得
$$
t(n)=1+\sum_{d=1}^n \mu(d)\,S\!\left(\left\lfloor\frac nd\right\rfloor\right),
$$
其中
$$
S(M)=\sum_{r=1}^M(3^r-2^r-1).
$$

$S(M)$ 的闭式为
$$
S(M)=\frac{3^{M+1}-3}{2}-(2^{M+1}-2)-M.
$$

最终任务是快速计算
$$
\sum_{d=1}^n \mu(d)\,S\!\left(\left\lfloor\frac nd\right\rfloor\right).
$$

做法：

- 用整除分块合并相同 $\left\lfloor n/d\right\rfloor$ 的区间。
- 需要频繁查询
  $$
  M(x)=\sum_{i=1}^x\mu(i),
  $$
  用杜教筛 + 记忆化计算。

按此实现可在 $n=10^{10}$ 下快速得到结果：
$$
t(10^{10})\bmod 10^9 = 268457129.
$$

## 代码

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

#ifdef LOCAL
#include "debug.h"
#else
#define debug(...) 42
#endif
using LL = long long;
using ULL = unsigned long long;
using i128 = __int128_t;

const LL MOD = 1000000000LL;
const LL MOD2 = 2000000000LL;

int lim;
vector<int> mu, pre_mu, primes;
vector<char> is_comp;
unordered_map<LL, LL> mertens_cache;

LL qpow(LL a, LL b, LL mod) {
	LL r = 1 % mod;
	while (b > 0) {
		if (b & 1) r = (LL)((i128)r * a % mod);
		a = (LL)((i128)a * a % mod);
		b >>= 1;
	}
	return r;
}

void sieve_mu(int n) {
	mu.assign(n + 1, 0);
	pre_mu.assign(n + 1, 0);
	is_comp.assign(n + 1, 0);
	primes.clear();

	mu[1] = 1;
	for (int i = 2; i <= n; ++i) {
		if (!is_comp[i]) {
			primes.push_back(i);
			mu[i] = -1;
		}
		for (int p : primes) {
			LL v = 1LL * i * p;
			if (v > n) break;
			is_comp[(int)v] = 1;
			if (i % p == 0) {
				mu[(int)v] = 0;
				break;
			} else {
				mu[(int)v] = -mu[i];
			}
		}
	}

	for (int i = 1; i <= n; ++i) pre_mu[i] = pre_mu[i - 1] + mu[i];
}

LL mertens(LL n) {
	if (n <= lim) return pre_mu[(int)n];
	auto it = mertens_cache.find(n);
	if (it != mertens_cache.end()) return it->second;

	LL res = 1;
	for (LL l = 2, r; l <= n; l = r + 1) {
		LL v = n / l;
		r = n / v;
		res -= (r - l + 1) * mertens(v);
	}
	return mertens_cache[n] = res;
}

LL S(LL m) {
	// (3^(m+1)-3)/2: first mod 2e9, then divide by 2.
	LL a = (qpow(3, m + 1, MOD2) - 3 + MOD2) % MOD2;
	a /= 2;

	LL b = (qpow(2, m + 1, MOD) - 2 + MOD) % MOD;

	LL ans = (a - b - (m % MOD) + 2) % MOD;
	if (ans < 0) ans += MOD;
	return ans;
}

void solve() {
	const LL n = 10000000000LL;

	lim = (int)(powl((long double)n, 2.0L / 3.0L) + 5);
	sieve_mu(lim);

	mertens_cache.reserve(1 << 20);
	mertens_cache.max_load_factor(0.7f);

	LL ans = 1;
	for (LL l = 1, r; l <= n; l = r + 1) {
		LL v = n / l;
		r = n / v;
		LL delta_mu = mertens(r) - mertens(l - 1);
		LL cur = S(v);
		ans = (ans + (LL)((i128)((delta_mu % MOD + MOD) % MOD) * cur % MOD)) % MOD;
	}

	cout << ans << '\n';
}

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	auto st = chrono::steady_clock::now();
	solve();
	auto ed = chrono::steady_clock::now();

	cerr << "Time: " << chrono::duration<double>(ed - st).count() << " s\n";
	return 0;
}
```

### Python

```python
import math
import sys
import time
from array import array

MOD = 1_000_000_000
MOD2 = 2_000_000_000


def sieve_mu(n: int):
	mu = array('i', [0]) * (n + 1)
	pre = array('i', [0]) * (n + 1)
	is_comp = bytearray(n + 1)
	primes = []

	mu[1] = 1
	for i in range(2, n + 1):
		if not is_comp[i]:
			primes.append(i)
			mu[i] = -1
		for p in primes:
			v = i * p
			if v > n:
				break
			is_comp[v] = 1
			if i % p == 0:
				mu[v] = 0
				break
			mu[v] = -mu[i]

	for i in range(1, n + 1):
		pre[i] = pre[i - 1] + mu[i]
	return pre


def solve() -> int:
	sys.setrecursionlimit(1_000_000)
	n = 10_000_000_000
	lim = int(math.pow(n, 2.0 / 3.0)) + 5

	pre_mu = sieve_mu(lim)
	cache = {}

	def mertens(x: int) -> int:
		if x <= lim:
			return pre_mu[x]
		if x in cache:
			return cache[x]

		res = 1
		l = 2
		while l <= x:
			v = x // l
			r = x // v
			res -= (r - l + 1) * mertens(v)
			l = r + 1
		cache[x] = res
		return res

	def S(m: int) -> int:
		a = (pow(3, m + 1, MOD2) - 3) % MOD2
		a //= 2
		b = (pow(2, m + 1, MOD) - 2) % MOD
		return (a - b - (m % MOD) + 2) % MOD

	ans = 1
	l = 1
	while l <= n:
		v = n // l
		r = n // v
		delta_mu = (mertens(r) - mertens(l - 1)) % MOD
		ans = (ans + delta_mu * S(v)) % MOD
		l = r + 1

	return ans


if __name__ == "__main__":
	st = time.perf_counter()
	ans = solve()
	ed = time.perf_counter()
	print(ans)
	print(f"Time: {ed - st:.3f} s", file=sys.stderr)
```

## 复杂度分析

时间复杂度：$O(N^{2/3})$。其中 $N=10^{10}$，主耗时来自 Möbius 前缀和的杜教筛计算，整除分块部分为 $O(\sqrt N)$ 段且每段只有常数次快速幂。

空间复杂度：$O(N^{2/3})$。主要由线性筛数组和杜教筛记忆化哈希表构成。

## 变形

### 变形 1：一般区间

整体框架不变，差异在于“完全 $d$ 次幂”的计数项需要改成区间 $(A^k,B^k)$ 下的对应整数计数。

### 变形 2：只求 $f(k)$

这正是中间量 $f(k)$，可直接用
$$
f(k)=\sum_{d\mid \operatorname{rad}(k)}\mu(d)\bigl(3^{k/d}-2^{k/d}-1\bigr)
$$
求值。

### 变形 3：一般代数约束

若把 $\alpha^k\in\mathbb{Z}$ 改成更一般的代数条件，通常不能直接沿用 Möbius + 完全幂容斥，需要新的结构化计数工具。
