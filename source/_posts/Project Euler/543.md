## 题意

定义函数 $P(n,k)$：

- 若 $n$ 可以表示成恰好 $k$ 个质数之和，则 $P(n,k)=1$；
- 否则 $P(n,k)=0$。

这里质数允许重复使用。

再定义：

$$
S(n)=\sum_{1\le i,k\le n} P(i,k)
$$

题目给出斐波那契数列 $F(k)$，其中：

$$
F(0)=0,\quad F(1)=1
$$

要求计算：

$$
\sum_{k=3}^{44} S(F(k))
$$

已知：

$$
S(10)=20,\quad S(100)=2402,\quad S(1000)=248838
$$

## 解题思路

### 第一步：$P(n,k)$ 在什么时候为 1

先按 $k$ 分类讨论。

#### 1. $k=1$

此时要求 $n$ 本身就是质数，所以：

$$
P(n,1)=1 \iff n\text{ 是质数}
$$

因此这部分对 $S(n)$ 的贡献就是质数计数函数：

$$
\pi(n)
$$

---

#### 2. $k=2$

如果两个质数之和是奇数，那么只能是：

$$
2 + (\text{奇质数})
$$

因此对奇数 $n$ 来说：

$$
P(n,2)=1 \iff n-2\text{ 是质数}
$$

而对偶数 $n\ge 4$，在本题需要的范围内都可以表示成两个质数之和，所以这部分可以直接计数。

于是当 $n\ge 4$ 时：

- 偶数部分有 $\left\lfloor \frac n2\right\rfloor-1$ 个；
- 奇数部分对应所有不超过 $n-2$ 的奇质数，个数是 $\pi(n-2)-1$。

所以：

$$
\begin{aligned}
B(n)
&= \left(\left\lfloor \frac n2\right\rfloor-1\right)+(\pi(n-2)-1) \\
&= \left\lfloor \frac n2\right\rfloor + \pi(n-2)-2
\end{aligned}
$$

---

#### 3. $k\ge 3$

先看必要条件。因为每个质数都至少是 2，所以 $k$ 个质数的和至少是：

$$
2k
$$

因此如果 $P(n,k)=1$，一定有：

$$
n\ge 2k
$$

下面证明这个条件也是充分的。

令：

$$
m=n-2(k-3)
$$

如果 $n\ge 2k$，那么：

$$
m\ge 2k-2(k-3)=6
$$

而每个不小于 6 的整数都可以表示成 3 个质数之和，于是 $m$ 可以写成 3 个质数之和，再补上 $k-3$ 个 2，就得到 $n$ 是 $k$ 个质数之和。

所以对所有 $k\ge 3$，有：

$$
P(n,k)=1 \iff n\ge 2k
$$

这时对于固定的 $i$，满足 $P(i,k)=1$ 的 $k$ 的范围就是：

$$
3\le k\le \left\lfloor \frac i2\right\rfloor
$$

因此这样的 $k$ 的个数为：

$$
\max\left(0,\left\lfloor \frac i2\right\rfloor-2\right)
$$

---

### 第二步：把 $S(n)$ 化成闭式

把所有贡献拆成三部分：

$$
S(n)=A(n)+B(n)+C(n)
$$

其中：

- $A(n)$ 对应 $k=1$
- $B(n)$ 对应 $k=2$
- $C(n)$ 对应 $k\ge 3$

前两部分已经得到：

$$
A(n)=\pi(n)
$$

以及当 $n\ge 4$ 时：

$$
B(n)=\left\lfloor \frac n2\right\rfloor+\pi(n-2)-2
$$

接下来处理：

$$
C(n)=\sum_{i=1}^{n}\max\left(0,\left\lfloor \frac i2\right\rfloor-2\right)
$$

当 $n\ge 6$ 时，上式可写成：

$$
\begin{aligned}
C(n)
&= \sum_{i=4}^{n}\left\lfloor \frac{i-4}{2}\right\rfloor \\
&= \sum_{j=0}^{n-4}\left\lfloor \frac j2\right\rfloor
\end{aligned}
$$

这里用到经典公式：

$$
\begin{aligned}
\sum_{j=0}^{m}\left\lfloor \frac{j}{2} \right\rfloor
&= \left\lfloor \frac{m^2}{4} \right\rfloor
\end{aligned}
$$

简单验证一下：

- 若 $m=2t$，则数列为 $0,0,1,1,\dots,t-1,t-1,t$，和为 $t^2$；
- 若 $m=2t+1$，则数列为 $0,0,1,1,\dots,t,t$，和为 $t(t+1)$。

而：

$$
\left\lfloor \frac{(2t)^2}{4}\right\rfloor=t^2,\qquad
\left\lfloor \frac{(2t+1)^2}{4}\right\rfloor=t(t+1)
$$

所以公式成立。

于是：

$$
C(n)=\left\lfloor \frac{(n-4)^2}{4}\right\rfloor
\qquad (n\ge 6)
$$

综上，$S(n)$ 可以写成分段公式：

$$
\begin{aligned}
S(n)
&= \pi(n) \\
&\quad + [n\ge 4]\cdot\left(\left\lfloor \frac n2 \right\rfloor + \pi(n-2) - 2\right) \\
&\quad + [n\ge 6]\cdot\left\lfloor \frac{(n-4)^2}{4} \right\rfloor
\end{aligned}
$$

这样问题就变成了：高效求很多次 $\pi(x)$。

---

### 第三步：用 Lehmer 算法求 $\pi(x)$

由于最大的查询值达到 $7\times 10^8$，直接筛到这个范围虽然也不是绝对不行，但没有必要。更自然的做法是使用 Lehmer prime counting 来计算质数个数。

设 $\phi(x,s)$ 表示 $1\sim x$ 中不被前 $s$ 个质数整除的整数个数，则 Lehmer 算法可以在较快时间内求出 $\pi(x)$。本题只需要对 $F(3)\sim F(44)$ 做有限次查询，再配合记忆化，运行速度足够快。

最后把所有 $S(F(k))$ 累加即可。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

#ifdef LOCAL
#include "debug.h"
#else
#define debug(...) 42
#endif
using LL = long long;
using ULL = unsigned long long;
using i128 = __int128_t;

vector<int> prm;
vector<int> piv;
vector<vector<int>> dp;
unordered_map<LL, LL> mp;

/* 求不大于 x 的整数平方根 */
LL isq(LL x) {
    LL r = sqrtl((long double)x);
    while ((r + 1) * (r + 1) <= x) r++;
    while (r * r > x) r--;
    return r;
}

/* 求不大于 x 的整数立方根 */
LL icb(LL x) {
    LL r = cbrtl((long double)x);
    while ((r + 1) * (r + 1) * (r + 1) <= x) r++;
    while (r * r * r > x) r--;
    return r;
}

/* 线性筛，预处理质数与 pi 数组 */
void siv(int n) {
    vector<int> lp(n + 1, 0);
    piv.assign(n + 1, 0);
    for (int i = 2; i <= n; i++) {
        if (!lp[i]) {
            lp[i] = i;
            prm.push_back(i);
        }
        for (int p : prm) {
            if (p > lp[i] || 1LL * i * p > n) break;
            lp[i * p] = p;
        }
        piv[i] = piv[i - 1] + (lp[i] == i);
    }
}

/* 求 phi(x, s)：1..x 中不被前 s 个质数整除的数的个数 */
LL phi(LL x, int s) {
    if (s == 0) return x;
    if (x < (int)dp.size() && s < (int)dp[0].size()) {
        return dp[(int)x][s];
    }
    return phi(x, s - 1) - phi(x / prm[s - 1], s - 1);
}

/* Lehmer 素数计数函数，返回 pi(x) */
LL cal(LL x) {
    if (x < (int)piv.size()) return piv[(int)x];
    if (mp.count(x)) return mp[x];

    LL a = cal(isq(isq(x))); // pi(x^(1/4))
    LL b = cal(isq(x));      // pi(x^(1/2))
    LL c = cal(icb(x));      // pi(x^(1/3))

    LL ans = phi(x, (int)a) + (b + a - 2) * (b - a + 1) / 2;

    for (LL i = a + 1; i <= b; i++) {
        LL w = x / prm[(int)i - 1];
        ans -= cal(w);
        if (i <= c) {
            LL lim = cal(isq(w));
            for (LL j = i; j <= lim; j++) {
                ans -= cal(w / prm[(int)j - 1]) - (j - 1);
            }
        }
    }

    mp[x] = ans;
    return ans;
}

/* 求 S(n) */
LL get(LL n) {
    LL ans = 0;

    // k = 1
    ans += cal(n);

    // k = 2
    if (n >= 4) {
        ans += n / 2 + cal(n - 2) - 2;
    }

    // k >= 3
    if (n >= 6) {
        ans += (n - 4) * (n - 4) / 4;
    }

    return ans;
}

/* 输出 i128 */
void pri(i128 x) {
    if (x == 0) {
        cout << 0 << '\n';
        return;
    }
    string s;
    while (x > 0) {
        s.push_back(char('0' + x % 10));
        x /= 10;
    }
    reverse(s.begin(), s.end());
    cout << s << '\n';
}

void solve() {
    auto st = chrono::steady_clock::now();

    int lim = 5000000;
    siv(lim);

    int m1 = 100000;
    int m2 = 100;
    dp.assign(m1, vector<int>(m2, 0));
    for (int i = 0; i < m1; i++) dp[i][0] = i;
    for (int j = 1; j < m2; j++) {
        for (int i = 0; i < m1; i++) {
            dp[i][j] = dp[i][j - 1] - dp[i / prm[j - 1]][j - 1];
        }
    }

    vector<LL> fib(45, 0);
    fib[0] = 0;
    fib[1] = 1;
    for (int i = 2; i <= 44; i++) {
        fib[i] = fib[i - 1] + fib[i - 2];
    }

    i128 ans = 0;
    for (int k = 3; k <= 44; k++) {
        ans += (i128)get(fib[k]);
    }

    pri(ans);

    auto ed = chrono::steady_clock::now();
    auto ms = chrono::duration_cast<chrono::milliseconds>(ed - st).count();
    cerr << "time = " << ms << " ms\n";
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    while (T--) {
        solve();
    }
    return 0;
}
```

## 复杂度分析

设本题中的最大查询值为 $N=F(44)$。

Lehmer 算法求单次 $\pi(x)$ 的复杂度通常可记为：

$$
O\left(\frac{x^{2/3}}{\log x}\right)
$$

本题只进行有限次查询，并且有记忆化复用，因此整体复杂度可以写成：

$$
O\left(M+\frac{N^{2/3}}{\log N}\right)
$$

其中 $M$ 是预筛上界，这份代码里取 $M=5\times 10^6$。

空间复杂度主要来自：

- 线性筛数组；
- $\phi$ 的小规模 DP 表；
- 质数表；
- 记忆化哈希表。

因此空间复杂度为：

$$
O(M+D)
$$

其中 $D$ 表示 $\phi$ 预处理表的规模。
