## 题目描述

题目链接：[Project Euler 457](https://projecteuler.net/problem=457)

定义

$$
f(n)=n^2-3n-1.
$$

对素数 $p$，令 $R(p)$ 为满足

$$
f(n)\equiv 0\pmod{p^2}
$$

的最小正整数 $n$；若不存在则定义 $R(p)=0$。

再定义

$$
SR(L)=\sum_{p\le L,\ p\text{ 为素数}}R(p).
$$

目标是计算 $SR(10^7)$。

## 解题思路

### 4.1 配方改写为平方同余

先做标准配方：

$$
4f(n)=4n^2-12n-4=(2n-3)^2-13.
$$

因此对奇素数 $p$，有

$$
f(n)\equiv 0\pmod{p^2}
\iff
(2n-3)^2\equiv 13\pmod{p^2}.
$$

问题本质变成：模 $p^2$ 下是否存在 $\sqrt{13}$，以及如何得到对应的最小正根。

### 4.2 特判 $p=2$ 与 $p=13$

先说明一个必要条件：若

$$
f(n)\equiv 0\pmod{p^2},
$$

则必有

$$
f(n)\equiv 0\pmod p.
$$

因此可先在模 $p$（或本小节中等价的模 $4$、模 $13$）下判断与构造。

#### 4.2.1 $p=2$：如何验证无解

此时需要

$$
f(n)=n^2-3n-1\equiv 0\pmod 4.
$$

模 $4$ 只需检查四个同余类 $n\equiv 0,1,2,3$：

$$
\begin{aligned}
f(0)&=-1\equiv 3\pmod 4,\\
f(1)&=-3\equiv 1\pmod 4,\\
f(2)&=-3\equiv 1\pmod 4,\\
f(3)&=-1\equiv 3\pmod 4.
\end{aligned}
$$

没有任何情形得到 $0\pmod 4$，故模 $4$ 无解，从而模 $2^2$ 无解，即

$$
R(2)=0.
$$

#### 4.2.2 $p=13$：如何得到重根 $8$

先在模 $13$ 下解

$$
f(n)\equiv 0\pmod{13}.
$$

利用

$$
4f(n)=(2n-3)^2-13,
$$

在模 $13$ 下化为

$$
(2n-3)^2\equiv 0\pmod{13}
\iff
2n-3\equiv 0\pmod{13}.
$$

又因为 $2^{-1}\equiv 7\pmod{13}$，所以

$$
n\equiv 3\cdot 2^{-1}\equiv 3\cdot 7\equiv 8\pmod{13}.
$$

因此模 $13$ 下唯一根是 $8$。它是重根：

$$
f(8)=39\equiv 0\pmod{13},\qquad f'(8)=2\cdot 8-3=13\equiv 0\pmod{13}.
$$

#### 4.2.3 为什么可以设 $n=8+13k$

如果某个整数 $n$ 满足模 $169$ 的同余

$$
f(n)\equiv 0\pmod{169},
$$

那么降模后必满足 $f(n)\equiv 0\pmod{13}$。而模 $13$ 下唯一根是 $8$，所以任何可能的候选都必须满足

$$
n\equiv 8\pmod{13}.
$$

同余类的标准写法就是

$$
n=8+13k\quad (k\in\mathbb{Z}).
$$

将其代回：

$$
\begin{aligned}
f(8+13k)
&=(8+13k)^2-3(8+13k)-1 \\
&=39+169k(k+1)
\equiv 39\not\equiv 0\pmod{169}.
\end{aligned}
$$

所以不存在可提升到模 $169$ 的根，结论为

$$
R(13)=0.
$$

### 4.3 判别式为 $0$ 为什么对应重根

一般地，在奇素数域 $\mathbb{F}_p$ 中，二次式

$$
ax^2+bx+c\equiv 0\pmod p\quad (a\not\equiv 0)
$$

的判别式为 $\Delta=b^2-4ac$。当 $\Delta\equiv 0\pmod p$ 时，根公式

$$
x\equiv\frac{-b\pm\sqrt{\Delta}}{2a}\pmod p
$$

中的 $\pm$ 两项重合，只剩一个根，即重根。

也可用导数判定：重根 $r$ 满足

$$
f(r)\equiv 0\pmod p,\qquad f'(r)\equiv 0\pmod p.
$$

本题在 $p=13$ 时正是该情形，所以不能直接套“普通根”的提升结论。

### 4.4 为什么只看 $p\bmod 13$

对 $p\neq 13$，可解性等价于

$$
\left(\frac{13}{p}\right)=1.
$$

这里的

$$
\left(\frac{a}{p}\right)
$$

是勒让德符号（$p$ 为奇素数）：

- $\left(\frac{a}{p}\right)=1$：$a$ 是模 $p$ 的平方剩余；
- $\left(\frac{a}{p}\right)=-1$：$a$ 不是模 $p$ 的平方剩余；
- $\left(\frac{a}{p}\right)=0$：$p\mid a$。

因此 $\left(\frac{13}{p}\right)=1$ 的含义就是：同余

$$
x^2\equiv 13\pmod p
$$

有解。

由于 $13\equiv 1\pmod 4$，二次互反律给出

$$
\left(\frac{13}{p}\right)=\left(\frac{p}{13}\right).
$$

二次互反律的一般形式（对不同奇素数 $\ell,m$）是

$$
\left(\frac{\ell}{m}\right)\left(\frac{m}{\ell}\right)
=(-1)^{\frac{\ell-1}{2}\cdot\frac{m-1}{2}}.
$$

常用等价写法（设 $p,q$ 为不同奇素数）：

- 若 $p\equiv 1\pmod 4$ 或 $q\equiv 1\pmod 4$，则

$$
\left(\frac{p}{q}\right)=\left(\frac{q}{p}\right);
$$

- 若 $p\equiv q\equiv 3\pmod 4$，则

$$
\left(\frac{p}{q}\right)=-\left(\frac{q}{p}\right).
$$

本题取 $\ell=13$，因为

$$
\frac{13-1}{2}=6
$$

是偶数，等价于 $13\equiv 1\pmod 4$，因此正好落在上面的第一种情形：可以直接交换而不变号。

而 $\left(\frac{p}{13}\right)$ 只由 $p\bmod 13$ 决定。模 $13$ 的非零平方剩余为

$$
\{1,3,4,9,10,12\},
$$

所以对 $p\neq 2,13$：

$$
R(p)\neq 0
\iff
p\bmod 13\in\{1,3,4,9,10,12\}.
$$

### 4.5 代换

设 $s$ 满足

$$
s^2\equiv 13\pmod p.
$$

由 $(2n-3)^2\equiv 13$，先令

$$
2n-3\equiv s\pmod p,
$$

得到

$$
r\equiv\frac{3+s}{2}\pmod p.
$$

这里的除以 $2$是模意义上的乘逆元：

$$
\frac{3+s}{2}\equiv (3+s)\cdot 2^{-1}\pmod p.
$$

因为 $p$ 为奇素数，$2^{-1}$ 一定存在。

### 4.6 Hensel 提升

已知 $r$ 是模 $p$ 的根，写作

$$
f(r)\equiv 0\pmod p.
$$

要找模 $p^2$ 的同余根，只需在同余类内修正：

$$
n=r+tp,
$$

其中 $t\in[0,p-1]$。代入

$$
f(r+tp)=f(r)+tp\,f'(r)+t^2p^2.
$$

上式可直接由代数展开得到。记 $h=tp$，则

$$
\begin{aligned}
f(r+h)
&=(r+h)^2-3(r+h)-1 \\
&=r^2+2rh+h^2-3r-3h-1 \\
&=(r^2-3r-1)+h(2r-3)+h^2 \\
&=f(r)+h\,f'(r)+h^2.
\end{aligned}
$$

把 $h=tp$ 代回即得

$$
f(r+tp)=f(r)+tp\,f'(r)+t^2p^2.
$$

因此导数 $f'(r)$ 出现的原因是：对增量 $h$ 的一次项系数正好是 $f'(r)$。对本题二次多项式而言，这个关系由展开式严格成立；从泰勒展开角度看，这正是一阶项。

令

$$
q=\frac{f(r)}{p},
$$

因为目标是让

$$
f(r+tp)\equiv 0\pmod{p^2},
$$

代入上式后得到

$$
f(r+tp)=pq+tp\,f'(r)+t^2p^2.
$$

所以条件等价于

$$
pq+tp\,f'(r)+t^2p^2\equiv 0\pmod{p^2}.
$$

提取一个 $p$：

$$
p\bigl(q+t f'(r)+tp\bigr)\equiv 0\pmod{p^2}.
$$

这等价于

$$
q+t f'(r)+tp\equiv 0\pmod p.
$$

又因为 $tp\equiv 0\pmod p$，于是得到一次同余

$$
q+t f'(r)\equiv 0\pmod p.
$$

于是

$$
t\equiv -q\,(f'(r))^{-1}\equiv -q\,(2r-3)^{-1}\pmod p.
$$

这就是从模 $p$ 到模 $p^2$ 的一步提升计算式。

### 4.7 为什么求出一个根后还不够

4.6 只构造出了一个模 $p^2$ 的根，但题目要求的是最小正根

注意到 $3-n$ 也是解。

$$
\begin{aligned}
f(3-n)
&=(3-n)^2-3(3-n)-1 \\
&=n^2-3n-1
=f(n).
\end{aligned}
$$

所以若

$$
f(n)\equiv 0\pmod{p^2},
$$

则必有

$$
f(3-n)\equiv 0\pmod{p^2}.
$$

设 4.6 得到的一支为 $n$，另一支就是 $3-n$。转成模 $p^2$ 的正代表，常写为

$$
n,\quad p^2+3-n.
$$

故最小正根为

$$
R(p)=\min\bigl(n,\ p^2+3-n\bigr).
$$

### 4.8 Tonelli-Shanks 算法

二次互反律只负责判断有无解，并不给出具体平方根。后续提升必须先拿到一个具体的

$$
s^2\equiv 13\pmod p,
$$

Tonelli-Shanks算法来构造 $s$。对应代码中的 `ts(13 % p, p)`。

### 4.9 Hensel定理

Hensel 的一般形式是从模 $p^k$ 提升到模 $p^{k+1}$。若已知

$$
f(a)\equiv 0\pmod{p^k},
$$

设新根为

$$
b=a+t p^k,
$$

可得

$$
t\equiv -\frac{f(a)}{p^k}(f'(a))^{-1}\pmod p.
$$

### 4.10 整体算法

1. 筛出所有 $p\le 10^7$ 的素数。
2. 对每个素数 $p$：
    - 若 $p=2$ 或 $p=13$，贡献为 $0$；
    - 若 $p\bmod 13\notin\{1,3,4,9,10,12\}$，贡献为 $0$；
    - 用 Tonelli-Shanks 求 $s^2\equiv 13\pmod p$；
    - 取 $r\equiv (3+s)\cdot 2^{-1}\pmod p$；
    - 用 Hensel 一步提升求 $t$，构造 $n=r+tp$；
    - 累加 $\min(n,p^2+3-n)$。
3. 用 `__int128` 累加并输出。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

using LL = long long;
using i128 = __int128_t;

const int LIM = 10000000;

LL qpm(LL a, LL e, LL mod) {
    LL r = 1 % mod;
    while (e) {
        if (e & 1) r = (LL)((i128)r * a % mod);
        a = (LL)((i128)a * a % mod);
        e >>= 1;
    }
    return r;
}

bool okm(int x) {
    return x == 1 || x == 3 || x == 4 || x == 9 || x == 10 || x == 12;
}

LL ts(LL n, LL p) {
    if (n == 0) return 0;
    if (p == 2) return n;
    if (qpm(n, (p - 1) / 2, p) != 1) return -1;
    if (p % 4 == 3) return qpm(n, (p + 1) / 4, p);

    LL q = p - 1, s = 0;
    while ((q & 1) == 0) {
        q >>= 1;
        ++s;
    }

    LL z = 2;
    while (qpm(z, (p - 1) / 2, p) != p - 1) ++z;

    LL c = qpm(z, q, p);
    LL x = qpm(n, (q + 1) / 2, p);
    LL t = qpm(n, q, p);
    LL m = s;

    while (t != 1) {
        LL i = 1;
        LL tt = (LL)((i128)t * t % p);
        while (tt != 1) {
            tt = (LL)((i128)tt * tt % p);
            ++i;
        }
        LL b = qpm(c, 1LL << (m - i - 1), p);
        x = (LL)((i128)x * b % p);
        c = (LL)((i128)b * b % p);
        t = (LL)((i128)t * c % p);
        m = i;
    }
    return x;
}

LL inv(LL a, LL p) {
    a %= p;
    if (a < 0) a += p;
    return qpm(a, p - 2, p);
}

LL calc_Rp(int p) {
    if (p == 2 || p == 13) return 0;
    if (!okm(p % 13)) return 0;

    LL s = ts(13 % p, p);
    if (s == -1) return 0;

    LL inv2 = (p + 1) / 2;
    LL r = (3 + s) % p * inv2 % p;

    LL fr = 1LL * r * r - 3LL * r - 1;
    LL q = fr / p;
    q %= p;
    if (q < 0) q += p;

    LL fp = (2LL * r - 3) % p;
    if (fp < 0) fp += p;
    LL t = (p - q) % p * inv(fp, p) % p;

    LL n = r + t * 1LL * p;
    LL m = 1LL * p * p + 3 - n;
    return min(n, m);
}

void print_i128(i128 x) {
    if (x == 0) {
        cout << 0;
        return;
    }
    if (x < 0) {
        cout << '-';
        x = -x;
    }
    string s;
    while (x) {
        s.push_back(char('0' + x % 10));
        x /= 10;
    }
    reverse(s.begin(), s.end());
    cout << s;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    auto bg = chrono::steady_clock::now();

    vector<bool> is_prime(LIM + 1, true);
    is_prime[0] = false;
    is_prime[1] = false;

    vector<int> primes;
    primes.reserve(700000);

    for (int i = 2; i <= LIM; ++i) {
        if (is_prime[i]) {
            primes.push_back(i);
            if (1LL * i * i <= LIM) {
                for (int j = i * i; j <= LIM; j += i) is_prime[j] = false;
            }
        }
    }

    i128 ans = 0;
    for (int p : primes) ans += calc_Rp(p);

    print_i128(ans);
    cout << '\n';

    auto ed = chrono::steady_clock::now();
    cerr << "Time: " << chrono::duration<double>(ed - bg).count() << " s\n";

    return 0;
}
```

## 复杂度分析

设 $L=10^7$，$\pi(L)$ 为不超过 $L$ 的素数个数。

时间复杂度：

$$
O\bigl(L\log\log L+\pi(L)\log^2 L\bigr).
$$

其中第一项来自筛素数，第二项来自每个候选素数上的模幂与 Tonelli-Shanks。

空间复杂度：

$$
O(L).
$$
