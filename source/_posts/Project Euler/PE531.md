## 题目描述

题目链接：[Project Euler 531](https://projecteuler.net/problem=531)

定义函数 $g(a,n,b,m)$ 为满足下列同余方程组的最小非负整数解：

$$
\begin{aligned}
x &\equiv a \pmod n,\\
x &\equiv b \pmod m.
\end{aligned}
$$

若该方程组无解，则定义 $g(a,n,b,m)=0$。

题目给出示例：

$$
g(2,4,4,6)=10,\qquad g(3,4,4,6)=0.
$$

记欧拉函数为 $\varphi(n)$，并定义

$$
f(n,m)=g\bigl(\varphi(n),n,\varphi(m),m\bigr).
$$

要求计算

$$
\sum_{1000000\le n<m<1005000} f(n,m).
$$

## 解题思路

### 4.1 广义 CRT

对固定一对 $(n,m)$，本题要求的是

$$
f(n,m)=g\bigl(\varphi(n),n,\varphi(m),m\bigr).
$$

记

$$
a=\varphi(n),\qquad b=\varphi(m).
$$

则等价于求同余方程组

$$
\begin{aligned}
x &\equiv a \pmod n,\\
x &\equiv b \pmod m.
\end{aligned}
$$

的最小非负解；若无解则记为 $0$。

因此本题的核心分成两步：

1. 判定何时有解。
2. 有解时构造最小非负解。

### 4.2 可解充要条件

命题：设 $a,b\in\mathbb Z$，$n,m\in\mathbb Z_{>0}$，记

$$
d=\gcd(n,m).
$$

考虑同余方程组

$$
\begin{aligned}
x &\equiv a \pmod n,\\
x &\equiv b \pmod m
\end{aligned}
$$

有解，当且仅当

$$
a\equiv b\pmod d.
$$

也即

$$
d\mid(a-b).
$$

证明分必要性与充分性两部分。

必要性：假设方程组有解，则存在整数 $x$ 使

$$
\begin{aligned}
x &= a+un,\\
x &= b+vm
\end{aligned}
$$

对某些整数 $u,v$ 成立。两式相减得到

$$
a-b=vm-un.
$$

右侧是 $n,m$ 的整数线性组合。由于 $d=\gcd(n,m)$，必有 $d\mid n$ 且 $d\mid m$，从而 $d$ 整除任意形如 $vm-un$ 的线性组合，所以

$$
d\mid(a-b),
$$

即 $a\equiv b\pmod d$。

充分性：现假设

$$
d\mid(a-b).
$$

目标是证明方程组一定有解。

第一步：化为线性方程。

设

$$
x=a+nt,
$$

其中 $t\in\mathbb Z$。代入第二个同余式：

$$
\begin{aligned}
a+nt &\equiv b \pmod m \\
nt &\equiv b-a \pmod m.
\end{aligned}
$$

这又等价于存在整数 $k$，使

$$
nt-mk=b-a.
$$

第二步：利用 $\gcd(n,m)=d$ 约化。

令

$$
\begin{aligned}
n &= dn_1,\\
m &= dm_1,\\
b-a &= dc,
\end{aligned}
$$

其中 $\gcd(n_1,m_1)=1$。将其代入 $nt-mk=b-a$ 并同除以 $d$，得到

$$
n_1t-m_1k=c.
$$

第三步：用裴蜀定理构造整数解。

由于 $\gcd(n_1,m_1)=1$，根据裴蜀定理，存在整数 $p,q$ 使

$$
n_1p+m_1q=1.
$$

两边乘以 $c$：

$$
n_1(cp)+m_1(cq)=c.
$$

改写为

$$
n_1(cp)-m_1(-cq)=c.
$$

取

$$
t=cp,\qquad k=-cq,
$$

即可得到 $n_1t-m_1k=c$ 的整数解，故 $nt-mk=b-a$ 有整数解，从而原同余方程组有解。

必要性与充分性均成立，命题得证。

因此本题中

$$
f(n,m)\ne 0\iff \varphi(n)\equiv\varphi(m)\pmod{\gcd(n,m)}.
$$

### 4.3 最小非负解的构造

当 4.2 的判定通过时，继续构造最小解。

记

$$
d=\gcd(n,m),\qquad n_1=\frac nd,\qquad m_1=\frac md,
$$

并令

$$
c=\frac{b-a}{d}.
$$

由上节推导，需要解

$$
n_1t\equiv c\pmod{m_1}.
$$

设逆元为 $n_1^{-1}\pmod{m_1}$，则

$$
t\equiv c\cdot n_1^{-1}\pmod{m_1}.
$$

取最小非负代表 $t_0\in[0,m_1-1]$，则

$$
x_0=a+nt_0.
$$

它就是所求的最小非负解。理由是：所有解都形如

$$
t=t_0+k m_1,
$$

从而

$$
\begin{aligned}
x &= a+n(t_0+k m_1) \\
    &= x_0+k\,n m_1 \\
    &= x_0+k\,\operatorname{lcm}(n,m).
\end{aligned}
$$

解集按固定正步长递增，因此 $k=0$ 对应最小非负值。

### 4.4 套回本题的逐对计算式

对每个 $1000000\le n<m<1005000$：

1. 设 $a=\varphi(n),\ b=\varphi(m),\ d=\gcd(n,m)$。
2. 若 $(b-a)\bmod d\ne 0$，则 $f(n,m)=0$。
3. 否则令

$$
\begin{aligned}
n_1 &= \frac nd,\\
m_1 &= \frac md,\\
c &= \frac{b-a}{d},
\end{aligned}
$$

求出

$$
t_0\equiv c\cdot n_1^{-1}\pmod{m_1},\quad 0\le t_0<m_1,
$$

并取

$$
f(n,m)=a+n t_0.
$$

### 4.5 欧拉函数的线性筛预处理

区间上界为 $1004999$，先一次性求出 $\varphi(1\sim 1004999)$。

在线性筛中维护质数表与最小质因子数组，转移为：

$$
\begin{aligned}
&\text{若 }i\text{ 为新质数，则 }\varphi(i)=i-1;\\
&\text{若 }p\mid i,\ \varphi(ip)=\varphi(i)\cdot p;\\
&\text{若 }p\nmid i,\ \varphi(ip)=\varphi(i)\cdot (p-1).
\end{aligned}
$$

这样可在线性时间内得到全部 $\varphi$ 值。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

#ifdef LOCAL
#include "debug.h"
#else
#define debug(...) 42
#endif
using LL = long long;
using ULL = unsigned long long;

const int L = 1000000;
const int R = 1004999;
const int U = R;

/* 求 a 在模 mod 下的逆元，要求 gcd(a, mod) = 1 */
LL inv(LL a, LL mod) {
    LL b = mod, x = 1, y = 0;
    while (b) {
        LL t = a / b;
        a -= t * b;
        swap(a, b);
        x -= t * y;
        swap(x, y);
    }
    x %= mod;
    if (x < 0) x += mod;
    return x;
}

/* 线性筛求 1..U 的欧拉函数 */
vector<int> get_phi() {
    vector<int> phi(U + 1), lp(U + 1), prm;
    phi[1] = 1;
    for (int i = 2; i <= U; i++) {
        if (!lp[i]) {
            lp[i] = i;
            prm.push_back(i);
            phi[i] = i - 1;
        }
        for (int p : prm) {
            LL v = 1LL * i * p;
            if (v > U) break;
            lp[v] = p;
            if (i % p == 0) {
                phi[v] = phi[i] * p;
                break;
            } else {
                phi[v] = phi[i] * (p - 1);
            }
        }
    }
    return phi;
}

void solve() {
    auto st = chrono::steady_clock::now();

    vector<int> phi = get_phi();
    LL ans = 0;

    for (int n = L; n <= R; n++) {
        int a = phi[n];
        for (int m = n + 1; m <= R; m++) {
            int b = phi[m];
            int d = gcd(n, m);
            int df = b - a;
            if (df % d != 0) continue;

            LL n1 = n / d;
            LL m1 = m / d;
            LL c = df / d;
            c %= m1;
            if (c < 0) c += m1;

            LL t = c * inv(n1, m1) % m1;
            ans += 1LL * a + 1LL * n * t;
        }
    }

    cout << ans << '\n';

    auto ed = chrono::steady_clock::now();
    double sec = chrono::duration<double>(ed - st).count();
    cerr << fixed << setprecision(6);
    cerr << "Elapsed: " << sec << " s\n";
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T = 1;
    while (T--) {
        solve();
    }
    return 0;
}
```

## 复杂度分析

设筛法上界为 $U=1004999$，区间长度为 $K=5000$。

时间复杂度：$O(U+K^2\log U)$。

说明如下：

- 线性筛为 $O(U)$。
- 数对枚举为 $O(K^2)$。
- 每对需要一次 $\gcd$ 和一次逆元计算，欧几里得过程为 $O(\log U)$。

空间复杂度：$O(U)$。

主要存储为 $\varphi$ 数组、筛法辅助数组与质数表。
