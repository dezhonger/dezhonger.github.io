## 题目描述

题目链接：[Project Euler 609](https://projecteuler.net/problem=609)

定义素数计数函数 $\pi(n)$ 为不超过 $n$ 的质数个数。一个整数序列 $u=(u_0,\dots,u_m)$ 称为 $\pi$ sequence，当且仅当：

1. 所有元素都满足 $u_i\ge 1$。
2. 对任意 $i<m$，有 $u_{i+1}=\pi(u_i)$。
3. 序列长度至少为 2。

记 $c(u)$ 为序列中非质数元素个数（注意 1 也算非质数）。
定义 $p(n,k)$ 为满足 $u_0\le n$ 且 $c(u)=k$ 的 $\pi$ sequence 数量。
再定义
$$
P(n)=\prod_{k:\,p(n,k)>0} p(n,k).
$$

已知 $P(10)=648$，$P(100)=31038676032$。要求计算
$$
P(10^8)\bmod 1000000007.
$$

## 解题思路

### 4.1 唯一链前缀化

固定起点 $x=u_0$ 后，后续项唯一确定：

$$
x\to \pi(x)\to \pi(\pi(x))\to \cdots
$$

因此从 $x$ 出发不存在分支搜索，只有一条主链。
所有合法 $\pi$ sequence 都是这条主链里长度至少为 2 的前缀。

因此问题可由枚举所有序列，转化为统计每条链前缀的属性分布。

### 4.2 定义 $H_m(y)$

$H_m(y)$ 用于压缩表示前缀统计信息。

先定义：$c(v)$ 表示前缀 $v$ 中非质数元素个数。对每个 $m\ge 1$，考虑链
$$
m,\ \pi(m),\ \pi^2(m),\ \dots,\ 1.
$$
记这条链所有非空前缀为 $v$，定义
$$
H_m(y)=\sum_{\text{prefix }v} y^{c(v)}.
$$

含义如下：

- 指数 $k$：非质数个数为 $k$。
- 系数：满足该非质数个数的前缀数量。

例如 $m=4$ 时，链为 $4\to2\to1$，前缀是
$(4)$、$(4,2)$、$(4,2,1)$，其非质数个数分别是 $1,1,2$，所以
$$
H_4(y)=2y+y^2.
$$
这就是将前缀统计写成多项式的形式。

### 4.3 前缀递推

设 $b(t)=1$ 表示 $t$ 非质数，$b(t)=0$ 表示 $t$ 是质数。

从 $m$ 出发的前缀分两类：

1. 只取 $(m)$。
2. 取 $(m)$ 后，再接上从 $\pi(m)$ 出发的某个非空前缀。

两类合并得到
$$
H_m(y)=y^{b(m)}\bigl(1+H_{\pi(m)}(y)\bigr).
$$

边界：
$$
H_1(y)=y,
$$
因为从 1 出发只有前缀 $(1)$，且 1 是非质数。

### 4.4 按 $\pi(x)=m$ 分组

设 $M=\pi(n)$。如果某个起点 $x$ 满足 $\pi(x)=m$，那么它对应序列的后缀结构完全由 $H_m$ 决定；
起点 $x$ 本身只影响是否额外增加 1 个非质数。

第 $m$ 组就是所有满足 $\pi(x)=m$ 且 $x\le n$ 的起点。

对 $m<M$，这组区间是
$$
[p_m,\ p_{m+1}-1],
$$
其中 $p_m$ 为第 $m$ 个质数。这个区间里恰好只有一个质数（就是 $p_m$），其余都是非质数。

于是组大小
$$
cnt_m=
\begin{cases}
p_{m+1}-p_m,&m<M,\\
n-p_M+1,&m=M.
\end{cases}
$$

第 $m$ 组对总生成函数的贡献就是
$$
\bigl(1+(cnt_m-1)y\bigr)H_m(y).
$$

### 4.5 总生成函数

总生成函数
$$
F_n(y)=\sum_{m=1}^{M}\bigl(1+(cnt_m-1)y\bigr)H_m(y).
$$

定义
$$
p(n,k)=[y^k]F_n(y),
$$
则题目所求
$$
P(n)=\prod_{k:\,p(n,k)>0}p(n,k)\pmod{1000000007}.
$$

### 4.6 系数 DP 与 $ans$ 转移

把 $H_m$ 的系数拿出来，定义
$$
dp[m][k]=[y^k]H_m(y).
$$
由 4.3 的递推直接得到：

- $m$ 是质数：
    $$
    dp[m][0]\mathrel{+}=1,\quad dp[m][k]\mathrel{+}=dp[\pi(m)][k].
    $$
- $m$ 非质数：
    $$
    dp[m][1]\mathrel{+}=1,\quad dp[m][k+1]\mathrel{+}=dp[\pi(m)][k].
    $$

接下来定义答案数组 $ans[k]=p(n,k)$。

固定某组 $m$，记 $c=cnt_m-1$（该组非质数起点个数）。

- 唯一质数起点：不会改变非质数个数，贡献
    $$
    ans[k]\mathrel{+}=dp[m][k].
    $$
- 其余 $c$ 个非质数起点：每条前缀都多一个非质数，贡献
    $$
    ans[k+1]\mathrel{+}=c\cdot dp[m][k].
    $$

这就是代码里两行更新 `ans` 的来源，它是
$$
\bigl(1+cy\bigr)H_m(y)
$$
的系数展开。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using LL = long long;

static constexpr int MOD = 1000000007;
static constexpr int K = 20;

// 只筛奇数：compOdd[x >> 1] = 1 表示奇数 x 是合数
vector<int> sievePrimes(int n, vector<unsigned char>& compOdd) {
    compOdd.assign((n >> 1) + 1, 0);
    vector<int> primes;
    if (n >= 2) primes.push_back(2);

    for (LL i = 3; i * i <= n; i += 2) {
        if (compOdd[i >> 1]) continue;
        for (LL j = i * i; j <= n; j += (i << 1)) {
            compOdd[j >> 1] = 1;
        }
    }
    for (int i = 3; i <= n; i += 2) {
        if (!compOdd[i >> 1]) primes.push_back(i);
    }
    return primes;
}

bool isPrime(int x, const vector<unsigned char>& compOdd) {
    if (x < 2) return false;
    if (x == 2) return true;
    if ((x & 1) == 0) return false;
    return !compOdd[x >> 1];
}

int solve(int n) {
    vector<unsigned char> compOdd;
    vector<int> primes = sievePrimes(n, compOdd);
    int M = (int)primes.size(); // M = pi(n)

    // piSmall[i] = pi(i), 只需要 i <= M
    vector<int> piSmall(M + 1, 0);
    int ptr = 0;
    for (int i = 1; i <= M; ++i) {
        while (ptr < (int)primes.size() && primes[ptr] <= i) ++ptr;
        piSmall[i] = ptr;
    }

    // dp[m][k]: 从 m 出发的非空前缀中，非质数数目为 k 的前缀个数
    vector<array<unsigned char, K>> dp(M + 1);
    for (int i = 0; i <= M; ++i) dp[i].fill(0);
    if (M >= 1) dp[1][1] = 1;

    vector<LL> ans(K, 0);

    for (int m = 1; m <= M; ++m) {
        if (m > 1) {
            int b = isPrime(m, compOdd) ? 0 : 1;
            int to = piSmall[m];

            // 前缀 (m)
            dp[m][b] += 1;

            // 前缀 (m) + prefix(from pi(m))
            for (int k = 0; k + b < K; ++k) {
                dp[m][k + b] += dp[to][k];
            }
        }

        int cnt = 0;
        if (m < M) cnt = primes[m] - primes[m - 1];
        else cnt = n - primes[m - 1] + 1;

        int compositeCount = cnt - 1;

        for (int k = 0; k < K; ++k) {
            if (dp[m][k]) ans[k] += dp[m][k];
            if (compositeCount && dp[m][k] && k + 1 < K) {
                ans[k + 1] += 1LL * compositeCount * dp[m][k];
            }
        }
    }

    LL result = 1;
    for (int k = 0; k < K; ++k) {
        if (ans[k] > 0) {
            result = (result * (ans[k] % MOD)) % MOD;
        }
    }
    return (int)result;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    auto start = chrono::steady_clock::now();

    const int n = 100000000;
    cout << solve(n) << '\n';

    auto end = chrono::steady_clock::now();
    double sec = chrono::duration<double>(end - start).count();
    cerr << "Time: " << fixed << setprecision(6) << sec << " s\n";
    return 0;
}
```

## 复杂度分析

设 $N=n$，$M=\pi(n)$，$K$ 为统计到的非质数个数上界（本题取常数 20）。

- 筛质数：$O(N\log\log N)$。
- 计算 $\pi(i)$（$1\le i\le M$）：$O(M)$。
- DP 与分组累加：$O(MK)$。

总时间复杂度：
$$
O(N\log\log N + MK).
$$

空间复杂度由筛数组和 DP 主导：
$$
O(N + MK).
$$
