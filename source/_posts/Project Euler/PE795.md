## 题意

题目链接：[Project Euler 795](https://projecteuler.net/problem=795)

定义

$$
g(n)=\sum_{i=1}^{n}(-1)^i\gcd(n,i^2).
$$

再定义前缀和

$$
G(N)=\sum_{n=1}^{N}g(n).
$$

已知：$g(4)=6$，$g(1234)=1233$，$G(1234)=2194708$。求

$$
G(12345678).
$$

## 解题思路

### 4.1 用欧拉函数拆 $\gcd$

经典恒等式：

$$
\gcd(a,b)=\sum_{d\mid a,\ d\mid b}\varphi(d).
$$

代入定义式：

$$
\begin{aligned}
g(n)
&=\sum_{i=1}^{n}(-1)^i\sum_{d\mid n,\ d\mid i^2}\varphi(d) \\
&=\sum_{d\mid n}\varphi(d)\sum_{\substack{1\le i\le n\\ d\mid i^2}}(-1)^i.
\end{aligned}
$$

于是问题变成：对固定约数 $d$，统计满足 $d\mid i^2$ 的项里，交错和 $\sum (-1)^i$ 到底是多少。

### 4.2 条件 $d\mid i^2$ 等价改写

设

$$
d=\prod p^{e_p}.
$$

条件 $d\mid i^2$ 等价于对每个质因子 $p$ 都有

$$
2v_p(i)\ge e_p\iff v_p(i)\ge\left\lceil\frac{e_p}{2}\right\rceil.
$$

定义

$$
t(d)=\prod p^{\lceil e_p/2\rceil},
$$

则

$$
d\mid i^2\iff t(d)\mid i.
$$

因此内层和可改写为

$$
\sum_{\substack{1\le i\le n\\ d\mid i^2}}(-1)^i
=\sum_{j=1}^{\lfloor n/t(d)\rfloor}(-1)^{j\,t(d)}.
$$

### 4.3 奇数 $n$ 的情况

当 $n$ 为奇数时，任意约数 $d\mid n$ 也为奇数，所以 $t(d)$ 为奇数。

此时

$$
(-1)^{j\,t(d)}=(-1)^j,
$$

且 $n/t(d)$ 也是奇数，因此

$$
\sum_{j=1}^{n/t(d)}(-1)^j=-1.
$$

代回后

$$
\begin{aligned}
g(n)
&=\sum_{d\mid n}\varphi(d)\cdot(-1) \\
&=-\sum_{d\mid n}\varphi(d)
=-n.
\end{aligned}
$$

所以得到关键结论：

$$
\boxed{n\text{ 为奇数时 }g(n)=-n}.
$$

### 4.4 偶数 $n$ 的情况

当 $n$ 为偶数：

- 若 $d$ 为奇约数，则 $t(d)$ 也是奇数，且 $n/t(d)$ 为偶数，交错和为 $0$；
- 若 $d$ 为偶约数，则 $t(d)$ 一定为偶数，故 $(-1)^{j\,t(d)}=1$，内层和就是项数

$$
\sum_{j=1}^{n/t(d)}1=\frac{n}{t(d)}.
$$

因此

$$
g(n)=\sum_{\substack{d\mid n\\ 2\mid d}}\varphi(d)\frac{n}{t(d)}.
$$

### 4.5 从 $g(n)$ 推到 $G(N)$

目标是

$$
G(N)=\sum_{n=1}^{N}g(n).
$$

按奇偶拆分：

$$
\begin{aligned}
G(N)
&=\sum_{\substack{1\le n\le N\\ n\text{ 奇}}}(-n)
+\sum_{\substack{1\le n\le N\\ n\text{ 偶}}}
\sum_{\substack{d\mid n\\ 2\mid d}}\varphi(d)\frac{n}{t(d)}.
\end{aligned}
$$

交换第二部分求和顺序。固定偶数 $d$，令 $n=dk$，其中

$$
1\le k\le\left\lfloor\frac{N}{d}\right\rfloor.
$$

则贡献为

$$
\sum_{k\le N/d}\varphi(d)\frac{dk}{t(d)}
=\varphi(d)\frac{d}{t(d)}\sum_{k\le N/d}k.
$$

记

$$
s(d)=\frac{d}{t(d)}=\prod p^{\lfloor e_p/2\rfloor},\qquad
M=\left\lfloor\frac{N}{d}\right\rfloor.
$$

则

$$
\begin{aligned}
G(N)
&=-\sum_{\substack{1\le n\le N\\ n\text{ 奇}}}n
+\sum_{\substack{2\le d\le N\\ d\text{ 偶}}}\varphi(d)s(d)\cdot\frac{M(M+1)}{2}.
\end{aligned}
$$

若奇数个数为

$$
m=\left\lfloor\frac{N+1}{2}\right\rfloor,
$$

则奇数和 $1+3+\cdots+(2m-1)=m^2$，最终得到

$$
\boxed{
G(N)=-m^2+\sum_{\substack{2\le d\le N\\ d\text{ 偶}}}\varphi(d)s(d)\cdot\frac{M(M+1)}{2}
}.
$$

### 4.6 在线性筛里维护 $s(n)$

设

$$
n=\prod p^{e_p},\qquad s(n)=\prod p^{\lfloor e_p/2\rfloor}.
$$

在线性筛中，令 $x=i\cdot p$：

- 若 $p\nmid i$，则质因子 $p$ 的指数从 $0$ 变成 $1$，$\lfloor1/2\rfloor=0$，故 $s(x)=s(i)$；
- 若 $p\mid i$，设 $p$ 在 $i$ 中指数为 $c$，新指数为 $c+1$：
  - 当 $c$ 为偶数，$\lfloor(c+1)/2\rfloor=\lfloor c/2\rfloor$，$s$ 不变；
  - 当 $c$ 为奇数，$\lfloor(c+1)/2\rfloor=\lfloor c/2\rfloor+1$，$s$ 额外乘一个 $p$。

这正对应代码里通过 `cnt` 的奇偶性更新 `sq` 的逻辑。最后枚举偶数 $d$ 累加公式即可得到答案。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

using LL = long long;
using i128 = __int128_t;

string to_string_i128(i128 x) {
    if (x == 0) return "0";
    bool neg = x < 0;
    if (neg) x = -x;
    string s;
    while (x > 0) {
        s.push_back(char('0' + x % 10));
        x /= 10;
    }
    if (neg) s.push_back('-');
    reverse(s.begin(), s.end());
    return s;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    const int N = 12345678;

    vector<int> lp(N + 1), phi(N + 1), cnt(N + 1), sq(N + 1), primes;
    primes.reserve(900000);

    phi[1] = 1;
    sq[1] = 1;

    for (int i = 2; i <= N; ++i) {
        if (!lp[i]) {
            lp[i] = i;
            primes.push_back(i);
            phi[i] = i - 1;
            cnt[i] = 1;
            sq[i] = 1;
        }

        for (int p : primes) {
            LL x = 1LL * i * p;
            if (x > N) break;

            lp[x] = p;
            if (i % p == 0) {
                phi[x] = phi[i] * p;
                cnt[x] = cnt[i] + 1;
                // cnt[i] 记录 i 的最小质因子幂次奇偶，决定 sq 是否要再乘 p。
                sq[x] = sq[i] * ((cnt[i] & 1) ? p : 1);
                break;
            } else {
                phi[x] = phi[i] * (p - 1);
                cnt[x] = 1;
                sq[x] = sq[i];
            }
        }
    }

    LL m = (N + 1LL) / 2;
    i128 ans = -(i128)m * m;

    for (int d = 2; d <= N; d += 2) {
        LL t = N / d;
        ans += (i128)phi[d] * sq[d] * t * (t + 1) / 2;
    }

    cout << to_string_i128(ans) << '\n';
    return 0;
}
```


## 复杂度分析

线性筛预处理 $\varphi$ 与 $sq$ 为 $O(N)$，最后遍历偶数 $d$ 累加也是 $O(N)$，总时间复杂度

$$
O(N).
$$

空间复杂度为若干个长度 $N$ 的数组：

$$
O(N).
$$

## 变形

- 把 $i^2$ 改成 $i^k$：只需把条件 $d\mid i^2$ 改成 $d\mid i^k$，对应阈值从 $\lceil e_p/2\rceil$ 变为 $\lceil e_p/k\rceil$。
- 去掉交错符号 $(-1)^i$：内层直接变为计数问题，整体仍可用“拆 $\gcd$ + 换序求和”的方法去做。
