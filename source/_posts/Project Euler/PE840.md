**题意**

题目链接：[Project Euler 840](https://projecteuler.net/problem=840)。

把正整数 $n$ 表示为若干正整数之和（不计顺序），称为一个 partition。定义函数 $D$：

$$
\begin{aligned}
D(1) &= 1, \\
D(p) &= 1 \quad (p\text{ 为质数}), \\
D(pq) &= D(p)q + pD(q) \quad (p,q>1).
\end{aligned}
$$

对分拆 $\{a_1,a_2,\dots,a_k\}$ 定义权值

$$
P=\prod_{j=1}^{k} D(a_j).
$$

再定义

$$
G(n)=\sum_{\text{n 的所有分拆}} P,
\qquad
S(N)=\sum_{n=1}^{N} G(n).
$$

已知 $G(10)=164$，$S(10)=396$。求

$$
S(5\times 10^4) \bmod 999676999.
$$

## 一、方法一（$O(N^2)$的DP）

### 1. 思路

定义

$$
G(n,k)=\text{把 }n\text{ 分拆且每个部件 }\le k\text{ 时，所有分拆权值之和}.
$$

那么原问题就是 $G(n)=G(n,n)$。

### 2. 推导

对固定的 $(n,k)$，所有分拆恰好分成两类：

1. 不使用 $k$：贡献是 $G(n,k-1)$。
2. 至少使用一个 $k$：去掉一个 $k$ 后，剩下是 $n-k$ 的分拆，且部件仍然不超过 $k$，贡献是 $D(k)\,G(n-k,k)$。

因此

$$
G(n,k)=G(n,k-1)+D(k)\,G(n-k,k).
$$

边界为

$$
G(0,k)=1,\qquad G(n,0)=0\ (n>0).
$$

这是一个带权完全背包。压成一维后用正向枚举：

$$
dp_s \mathrel{+}= D(k)\,dp_{s-k}\quad (s=k\dots N).
$$

处理完第 $k$ 层后，$dp_s=G(s,k)$；最终 $dp_n=G(n,n)=G(n)$，再做前缀和得到 $S(N)$。

### 3. 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

using LL = long long;
const int MOD = 999676999;

vector<int> calcD(int n) {
    vector<int> d(n + 1), lp(n + 1), primes;
    d[1] = 1;
    for (int i = 2; i <= n; ++i) {
        if (!lp[i]) {
            lp[i] = i;
            primes.push_back(i);
            d[i] = 1;
        }
        for (int p : primes) {
            if (1LL * i * p > n) break;
            int v = i * p;
            lp[v] = p;
            d[v] = (1LL * p * d[i] + i) % MOD;
            if (i % p == 0) break;
        }
    }
    return d;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    const int N = 50000;
    auto st = chrono::steady_clock::now();

    vector<int> d = calcD(N);
    vector<int> dp(N + 1, 0);
    dp[0] = 1;

    for (int k = 1; k <= N; ++k) {
        for (int s = k; s <= N; ++s) {
            dp[s] = (dp[s] + 1LL * d[k] * dp[s - k]) % MOD;
        }
    }

    int ans = 0;
    for (int n = 1; n <= N; ++n) {
        ans += dp[n];
        if (ans >= MOD) ans -= MOD;
    }

    cout << ans << '\n';

    auto ed = chrono::steady_clock::now();
    auto ms = chrono::duration_cast<chrono::milliseconds>(ed - st).count();
    cerr << "Time: " << ms << " ms\n";
    return 0;
}
```

### 4. 复杂度分析

时间复杂度：$O(N^2)$。  
空间复杂度：$O(N)$。

## 二、方法二（生成函数与快速卷积）

### 1. 思路

1. 先把题目改写成生成函数；
2. 再把生成函数推成卷积递推；
3. 最后用 CDQ + NTT + CRT 优化。

### 2. 推导

#### 3.1 先把 $D(n)$ 看明白

由定义：

$$
D(ab)=D(a)b+aD(b)
$$

这和普通求导的乘法法则一致。

先看质数幂。对质数 $p$，因为 $D(p)=1$：

$$
\begin{aligned}
D(p^2)&=D(p)p+pD(p)=2p, \\
D(p^3)&=D(p^2)p+p^2D(p)=3p^2.
\end{aligned}
$$

归纳可得

$$
D(p^e)=e p^{e-1}.
$$

再看一般整数。若

$$
n=\prod_{i=1}^{m} p_i^{e_i},
$$

由乘法法则展开：

$$
D(n)=\sum_{i=1}^{m} D(p_i^{e_i})\prod_{j\ne i}p_j^{e_j}.
$$

代入 $D(p_i^{e_i})=e_i p_i^{e_i-1}$：

$$
D(n)=\sum_{i=1}^{m} e_i p_i^{e_i-1}\prod_{j\ne i}p_j^{e_j}
=n\sum_{i=1}^{m}\frac{e_i}{p_i}.
$$

#### 3.2 分拆问题的标准思路：生成函数

设

$$
F(x)=\sum_{n\ge 0} G(n)x^n.
$$

对每个部件值 $m$，它可以出现 $0,1,2,\dots$ 次：

1. 出现 0 次，贡献 $1$；
2. 出现 1 次，贡献 $D(m)x^m$；
3. 出现 2 次，贡献 $D(m)^2x^{2m}$；
4. 出现 3 次，贡献 $D(m)^3x^{3m}$。

所以关于 $m$ 的局部生成函数是

$$
1+D(m)x^m+D(m)^2x^{2m}+\cdots
=\frac{1}{1-D(m)x^m}.
$$

所有 $m$ 独立，因此

$$
F(x)=\prod_{m\ge 1}\frac{1}{1-D(m)x^m}.
$$

#### 3.3 从生成函数推递推式

对 $F(x)$ 取对数：

$$
\ln F(x)= -\sum_{m\ge 1}\ln(1-D(m)x^m).
$$

利用

$$
-\ln(1-y)=\sum_{r\ge 1}\frac{y^r}{r},
$$

得到

$$
\ln F(x)=\sum_{m\ge 1}\sum_{r\ge 1}\frac{D(m)^r x^{mr}}{r}.
$$

对两边求导再乘 $x$：

$$
\frac{xF'(x)}{F(x)}=\sum_{m\ge 1}\sum_{r\ge 1} mD(m)^r x^{mr}.
$$

按同次幂合并，定义

$$
A(n)=\sum_{d\mid n} d\,D(d)^{n/d},
$$

则

$$
\frac{xF'(x)}{F(x)}=\sum_{n\ge 1} A(n)x^n.
$$

于是

$$
xF'(x)=F(x)\sum_{n\ge 1}A(n)x^n.
$$

记

$$
F(x)=\sum_{n\ge 0} g_nx^n,\quad g_n=G(n),\quad g_0=1,
$$

有

$$
xF'(x)=\sum_{n\ge 1} ng_nx^n.
$$

比较 $x^n$ 系数：

$$
ng_n=\sum_{k=1}^{n}A(k)g_{n-k}.
$$

所以核心递推为

$$
g_0=1,\qquad g_n=\frac{1}{n}\sum_{k=1}^{n}A(k)g_{n-k}.
$$

#### 3.4 如何计算 $A(n)$

由定义

$$
A(n)=\sum_{d\mid n} d\,D(d)^{n/d}.
$$

设 $n=td$。固定 $d$ 时，它对 $d,2d,3d,\dots$ 依次贡献

$$
d\,D(d)^1,\ d\,D(d)^2,\ d\,D(d)^3,\dots
$$

做法是：

1. 固定 $d$；
2. 维护 $cur=D(d)^t$；
3. 枚举 $m=d,2d,3d,\dots$，把 $d\cdot cur$ 加到 $A(m)$。

复杂度：

$$
\sum_{d=1}^{N}\frac{N}{d}=O(N\log N).
$$

#### 3.5 如何计算 $D(n)$

利用

$$
D(ip)=pD(i)+i\qquad (p\text{ 是质数}).
$$

这是由

$$
D(ip)=D(i)\cdot p+i\cdot D(p)=pD(i)+i
$$

直接得到的。在线性筛中可顺便算出所有 $D(n)$，复杂度 $O(N)$。

#### 3.6 用 CDQ 分治优化卷积

直接按递推

$$
g_n=\frac{1}{n}\sum_{k=1}^{n}A(k)g_{n-k}
$$

计算，复杂度是 $O(N^2)$。当 $N=50000$ 时，转移次数约为

$$
\sum_{n=1}^{N}n=\frac{N(N+1)}2=1{,}250{,}025{,}000,
$$

带模乘加后常数较大。

递推本质是“已知前缀 $g_0,\dots,g_{n-1}$ 求 $g_n$”的卷积型转移，适合 CDQ。

处理区间 $[l,r]$，设中点 $mid$：

1. 先递归算出 $[l,mid]$ 的 $g$；
2. 用它批量更新 $(mid,r]$；
3. 批量更新就是卷积：左侧 $g[l\dots mid]$，右侧 $A[1\dots r-l]$。

每层都可用快速卷积完成。由于模数 $999676999$ 不是常见 NTT 模数，代码里使用三模数 NTT 后 CRT 合并。

### 3. 算法流程总结

1. 线性筛，求出所有 $D(1\ldots N)$；
2. 枚举约数，求出所有 $A(1\ldots N)$；
3. 用 CDQ 分治 + NTT + CRT 计算所有 $g_n=G(n)$；
4. 最后求前缀和得到 $S(N)$。

### 4. 补充

#### 4.1 为什么局部是 $D(m)^t$

因为分拆权值定义为

$$
P=\prod_j D(a_j).
$$

若某个数 $m$ 出现 $t$ 次，乘积里就会出现 $t$ 个 $D(m)$，即 $D(m)^t$。

#### 4.2 为什么 $[x^n]F(x)=G(n)$

展开乘积时，对每个 $m$ 选择出现次数 $t_m$：

1. 指数 $\sum mt_m$ 决定和为多少；
2. 系数 $\prod D(m)^{t_m}$ 就是这组分拆的权值。

因此 $x^n$ 的系数恰好是“所有和为 $n$ 的分拆权值之和”，即 $G(n)$。

#### 4.3 有了 $F(x)$ 后，$S(N)$ 怎么看

由

$$
F(x)=\sum_{n\ge 0}G(n)x^n
$$

可知 $G(n)$ 是 $x^n$ 的系数，所以

$$
S(N)=\sum_{n=1}^{N}G(n)
$$

就是前 $N$ 个非零次幂系数之和。

### 5. 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

using LL = long long;
using i128 = __int128_t;

const int MOD = 999676999;
const int M1 = 998244353;
const int M2 = 1004535809;
const int M3 = 469762049;
const int G1 = 3;
const int G2 = 3;
const int G3 = 3;

LL pw(LL a, LL b, LL mod) {
    LL r = 1 % mod;
    while (b) {
        if (b & 1) r = r * a % mod;
        a = a * a % mod;
        b >>= 1;
    }
    return r;
}

template<int P, int G>
struct Ntt {
    vector<int> rt, rv;

    void ntt(vector<int>& a, bool iv) {
        int n = (int)a.size();
        if ((int)rv.size() != n) {
            rv.assign(n, 0);
            int k = __builtin_ctz(n);
            for (int i = 0; i < n; i++) {
                rv[i] = (rv[i >> 1] >> 1) | ((i & 1) << (k - 1));
            }
        }
        if ((int)rt.size() < n) {
            int k = rt.empty() ? 1 : __builtin_ctz((int)rt.size());
            if (rt.empty()) rt = {0, 1};
            while ((1 << k) < n) {
                int e = (int)pw(G, (P - 1) >> (k + 1), P);
                rt.resize(1 << (k + 1));
                for (int i = 1 << (k - 1); i < (1 << k); i++) {
                    rt[i << 1] = rt[i];
                    rt[i << 1 | 1] = (LL)rt[i] * e % P;
                }
                k++;
            }
        }

        for (int i = 0; i < n; i++) {
            if (i < rv[i]) swap(a[i], a[rv[i]]);
        }

        for (int m = 1; m < n; m <<= 1) {
            for (int i = 0; i < n; i += (m << 1)) {
                for (int j = 0; j < m; j++) {
                    int x = a[i + j];
                    int y = (LL)a[i + j + m] * rt[m + j] % P;
                    a[i + j] = x + y < P ? x + y : x + y - P;
                    a[i + j + m] = x - y >= 0 ? x - y : x - y + P;
                }
            }
        }

        if (iv) {
            reverse(a.begin() + 1, a.end());
            int in = (int)pw(n, P - 2, P);
            for (int& x : a) x = (LL)x * in % P;
        }
    }

    vector<int> mul(const vector<int>& a, const vector<int>& b) {
        if (a.empty() || b.empty()) return {};
        int s = (int)a.size() + (int)b.size() - 1;
        int n = 1;
        while (n < s) n <<= 1;

        vector<int> f(a.begin(), a.end()), g(b.begin(), b.end());
        f.resize(n);
        g.resize(n);

        ntt(f, false);
        ntt(g, false);
        for (int i = 0; i < n; i++) {
            f[i] = (LL)f[i] * g[i] % P;
        }
        ntt(f, true);
        f.resize(s);
        return f;
    }
};

Ntt<M1, G1> n1;
Ntt<M2, G2> n2;
Ntt<M3, G3> n3;

vector<int> a, g, iv;

vector<int> mul(const vector<int>& x, const vector<int>& y) {
    auto a1 = n1.mul(x, y);
    auto a2 = n2.mul(x, y);
    auto a3 = n3.mul(x, y);

    int n = (int)a1.size();
    vector<int> r(n);

    const LL m1 = M1, m2 = M2, m3 = M3;
    const LL im1m2 = pw(m1 % m2, m2 - 2, m2);
    const LL m1m3 = m1 % m3;
    const LL m12m3 = (i128)m1m3 * (m2 % m3) % m3;
    const LL im12m3 = pw(m12m3, m3 - 2, m3);

    const LL m1md = m1 % MOD;
    const LL m12md = (i128)(m1 % MOD) * (m2 % MOD) % MOD;

    for (int i = 0; i < n; i++) {
        LL x1 = a1[i];
        LL x2 = a2[i];
        LL x3 = a3[i];

        LL t1 = (x2 - x1) % m2;
        if (t1 < 0) t1 += m2;
        t1 = t1 * im1m2 % m2;

        LL x12m3 = (x1 + (i128)m1m3 * t1) % m3;

        LL t2 = (x3 - x12m3) % m3;
        if (t2 < 0) t2 += m3;
        t2 = t2 * im12m3 % m3;

        LL ans = (x1 % MOD + (i128)m1md * (t1 % MOD) + (i128)m12md * (t2 % MOD)) % MOD;
        r[i] = (int)ans;
    }
    return r;
}

void cdq(int l, int r) {
    if (l == r) {
        if (l > 0) g[l] = (LL)g[l] * iv[l] % MOD;
        return;
    }
    int m = (l + r) >> 1;
    cdq(l, m);

    vector<int> x(m - l + 1), y(r - l);
    for (int i = l; i <= m; i++) x[i - l] = g[i];
    for (int i = 1; i <= r - l; i++) y[i - 1] = a[i];

    auto z = mul(x, y);

    for (int i = m + 1; i <= r; i++) {
        int p = i - l - 1;
        if (0 <= p && p < (int)z.size()) {
            g[i] += z[p];
            if (g[i] >= MOD) g[i] -= MOD;
        }
    }

    cdq(m + 1, r);
}

void solve() {
    const int n = 50000;

    vector<int> d(n + 1), sp(n + 1), pr;
    d[1] = 1;

    for (int i = 2; i <= n; i++) {
        if (!sp[i]) {
            sp[i] = i;
            pr.push_back(i);
            d[i] = 1;
        }
        for (int p : pr) {
            LL v = 1LL * i * p;
            if (v > n) break;
            sp[v] = p;
            d[v] = ((LL)p * d[i] + i) % MOD;
            if (i % p == 0) break;
        }
    }

    a.assign(n + 1, 0);
    for (int x = 1; x <= n; x++) {
        int cur = 1;
        int dx = d[x];
        for (int y = x; y <= n; y += x) {
            cur = (LL)cur * dx % MOD;
            a[y] += (LL)x * cur % MOD;
            if (a[y] >= MOD) a[y] -= MOD;
        }
    }

    iv.assign(n + 1, 1);
    for (int i = 2; i <= n; i++) {
        iv[i] = MOD - (LL)(MOD / i) * iv[MOD % i] % MOD;
    }

    g.assign(n + 1, 0);
    g[0] = 1;

    cdq(0, n);

    int ans = 0;
    for (int i = 1; i <= n; i++) {
        ans += g[i];
        if (ans >= MOD) ans -= MOD;
    }

    cout << ans << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    auto st = chrono::steady_clock::now();

    solve();

    auto ed = chrono::steady_clock::now();
    auto ms = chrono::duration_cast<chrono::milliseconds>(ed - st).count();
    cerr << "Time: " << ms << " ms\n";

    return 0;
}
```

### 6. 复杂度分析

1. 线性筛 $D$：$O(N)$；
2. 预处理 $A$：$O(N\log N)$；
3. CDQ + NTT + CRT：$O(N\log^2 N)$。

因此该实现的总时间复杂度为

$$
O(N\log^2 N),
$$

空间复杂度为

$$
O(N).
$$

**变形**

1. 若把每个部件权值从 $D(m)$ 改为任意函数 $w(m)$，则只需把

$$
F(x)=\prod_{m\ge 1}\frac{1}{1-w(m)x^m}
$$

代入同样流程。

2. 若限制“每个数最多出现一次”，则局部因子改为 $1+w(m)x^m$，对应 0/1 版本。

3. 若要求“恰好拆成 $k$ 项”，可再引入变量 $y$ 记录项数：

$$
\prod_{m\ge 1}\frac{1}{1-y\,w(m)x^m},
$$

最后取 $y^k$ 系数。
